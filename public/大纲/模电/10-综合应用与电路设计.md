### 2. 电路选型对照表（综合设计题 1）

题干给出十种电路（A–J），按功能选型：

| 功能需求 | 选型 | 关键理由 |
|----------|------|----------|
| $Y = aX_1 + bX_2 + cX_3$，$a>0, abc<0$ | **I 加减运算电路** | 既有加（同相端）又有减（反相端），满足正负混合系数 |
| $A_v > 100$ 放大 | **A 反相比例** 或 B 同相比例 | 反相 $A_v = -R_f/R_1$；同相 $A_v = 1+R_f/R_1$ |
| 正弦波**移相 +90°** | **D 积分运算电路** | 反相积分对 $\sin(\omega t)$ 输出 $\cos(\omega t)$，超前 90° |
| 方波转**尖顶波** | **E 微分运算电路** | 微分对阶跃边沿产生冲激 |
| $A_v = 1$ | **J 电压跟随器** | $v_o = v_i$，高 Ri 低 Ro |

十种电路清单：A 反相比例、B 同相比例、C 差分比例、D 积分、E 微分、F 反相求和、G 同相求和、H 乘方、I 加减、J 电压跟随器。

> ⚠️ **修正**：$A_v > 100$ 答案为 **A 反相比例放大电路**（答案明确），同相比例亦可但反相更常用。

### 3. 反相加法器设计（综合设计题 2，教材原题 2.4.1）⭐ 修正

**已知**：设计反相加法器，最大电阻 ≤ 280 kΩ，输出 $v_o = -(7v_{i1} + 14v_{i2} + 3.5v_{i3} + 10v_{i4})$

**输出关系**：

$$v_o = -\left(\frac{R_f}{R_1}v_{i1} + \frac{R_f}{R_2}v_{i2} + \frac{R_f}{R_3}v_{i3} + \frac{R_f}{R_4}v_{i4}\right)$$

各支路增益系数须满足 $R_f/R_i$ = 对应系数。

**关键：取 $R_f$ 为系数的最小公倍数**

系数 7, 14, 3.5, 10 的最小公倍数为 **140**，故取：

$$\boxed{R_f = 140\,\text{k}\Omega}$$

> ⚠️ **修正**：旧版大纲误写 $R_f = 280\,\text{k}\Omega$。答案明确取 **140 kΩ**（系数 LCM），使各支路电阻均为整数标准值。

| 输入 | 系数 | 电阻值 | 计算 |
|------|------|--------|------|
| $v_{i1}$ | 7 | $R_1 = 140/7$ | **20 kΩ** |
| $v_{i2}$ | 14 | $R_2 = 140/14$ | **10 kΩ** |
| $v_{i3}$ | 3.5 | $R_3 = 140/3.5$ | **40 kΩ** |
| $v_{i4}$ | 10 | $R_4 = 140/10$ | **14 kΩ** |

校核：所有电阻 ≤ 280 kΩ（最大为 $R_f = 140\,\text{k}\Omega$）✓

### 4. 智能安防监测系统（综合设计题 3）

**任务链**：传感器（5mV / 1kHz）→ 同相放大 → 低通滤波 → AI 决策

- **放大电路**：5mV → 1V，增益 $A_v = 1\text{V}/5\text{mV} = 200$
  - 采用同相比例：$A_v = 1 + R_f/R_1 = 200$，取 $R_1 = 1\,\text{k}\Omega$，$R_f = 199\,\text{k}\Omega$
- **一阶低通滤波器**：截止频率 $f_c = 5\,\text{kHz}$
  - $f_c = 1/(2\pi RC)$，取 $C = 0.01\,\mu\text{F}$，则 $R \approx 3.2\,\text{k}\Omega$

### 5. 智能义肢力反馈系统（综合设计题 4）⭐ 修正

**任务链**：压力传感器（0~80mV 差分）→ 差分放大 → 一阶 RC 低通（$f_c$=30Hz）→ AI 决策

#### 5.1 差分放大电路设计

- 增益：$A_u = 4\text{V}/80\text{mV} = 50$
- 已知 $R_1 = R_2 = 5.1\,\text{k}\Omega$

$$R_f = R_g = A_u \times R_1 = 50 \times 5.1\,\text{k}\Omega = \boxed{255\,\text{k}\Omega}$$

#### 5.2 一阶 RC 低通滤波器设计 ⚠️ 修正

已知 $f_c = 30\,\text{Hz}$，$C = 2.2\,\mu\text{F}$：

$$R = \frac{1}{2\pi f_c C} = \frac{1}{2\pi \times 30 \times 2.2 \times 10^{-6}} \approx \boxed{2.4\,\text{k}\Omega}$$

> ⚠️ **修正**：旧版大纲写"参考答案按 16Hz 计算"有误。答案明确 $f_c = 30\,\text{Hz}$，$R \approx 2.4\,\text{k}\Omega$。

**滤波效果**：600Hz 干扰衰减 $1/\sqrt{1+(600/30)^2} \approx 0.05$（约 −26dB），残余约 2mV，有效抑制。

### 6. AI 四状态决策表（义肢案例）

| 状态 | 力信号斜率特征 | 电机控制策略 |
|------|----------------|--------------|
| **尚未接触** | 压力≈0，斜率=0 | 保持手指张开，等待抓取 |
| **轻柔抓取** | 斜率小且稳，稳定在低压力区 | 低速闭合，达到设定力后停机 |
| **抓取过紧** | 斜率偏大，力超安全阈值 | 反转松开，回调至标准范围 |
| **物体滑脱** | 稳定压力**快速下降**，斜率负且大 | 立即正转加大力；多次失败则报警 |

## 重点公式 / 结论速记

| 项目 | 公式/结论 |
|------|----------|
| 同相比例增益 | $A_v = 1 + R_f/R_1$ |
| 反相比例增益 | $A_v = -R_f/R_1$ |
| 反相加法器支路电阻 | $R_i = R_f / \text{系数}$，$R_f$ = 系数的 LCM |
| 差分放大增益 | $A_u = R_f/R_1$ |
| 一阶 RC 低通截止频率 | $f_c = 1/(2\pi RC)$ |
| 共模抑制比 | $CMRR = |A_d/A_c|$ |
| 理想运放两特性 | 虚短、虚断（前提：深度负反馈 + 线性区） |
| 电压串联负反馈 | 提高输入电阻 + 稳定输出电压 |
| 移相 +90° | 用**积分**电路（不是微分） |
| 方波转尖顶波 | 用**微分**电路 |
