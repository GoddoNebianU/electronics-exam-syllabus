## 核心知识点

### 1. BJT 与 FET 的本质区别（填空 19–20、25–27）

| 对比项 | BJT 双极型三极管 | FET 场效应管 |
|--------|------------------|--------------|
| 控制方式 | **电流**控制器件 | **电压**控制器件 |
| 载流子 | **两**种（电子+空穴） | 一种（多子） |
| 输入电阻 | 小 | **大** |
| 控制量 | $i_B$ 控制 $i_C$ | $u_{GS}$ 控制 $i_D$ |
| 分类 | NPN / PNP | 结型 / 绝缘栅（MOS） |

- **填空19**：BJT 是**电流**控制器件，FET 是**电压**控制器件
- **填空20**：BJT 有**两**种载流子参与导电
- **填空25**：FET 分为结型和**绝缘栅（MOS）**场效应管；输入电阻**高**
- **填空26**：FET 是**电压**控制器件，栅源电压控制**漏极**电流

### 2. BJT 三种组态与三工作区（填空 21、23–24）

#### 2.1 三种组态（填空 23）

- **共发射极**（CE）、**共集电极**（CC）、**共基极**（CB）

#### 2.2 输出特性三区（填空 24）

| 区域 | 偏置条件（NPN） | 特点 |
|------|-----------------|------|
| **截止区** | 发射结反偏、集电结反偏 | $I_C \approx 0$ |
| **放大区** | 发射结**正偏**、集电结**反偏** | $I_C = \beta I_B$，线性放大 |
| **饱和区** | 发射结正偏、集电结正偏 | $V_{CES} \approx 0.3\,\text{V}$ |

- **填空20–21**：放大区偏置条件：发射结**正向偏置（正偏）**、集电结**反向偏置（反偏）**

### 3. 复合管（填空 22）

- 两 β 相同晶体管组成复合管（达林顿管）：电流放大系数 $\beta \approx \beta_1 \cdot \beta_2$（即 $\beta^2$）

### 4. 静态工作点与非线性失真（填空 28–30）

| Q 点位置 | 失真类型 | 输出波形 | 消除方法 |
|----------|----------|----------|----------|
| **偏高**（接近饱和） | **饱和失真** | 输出**底部削平** | **增大**基极偏置电阻 $R_b$ |
| **偏低**（接近截止） | **截止失真** | 输出**顶部削平** | **减小**基极偏置电阻 $R_b$ |

- **填空28**：Q 点偏高→**饱和**失真；过低→**截止**失真
- **填空29**：输出顶部削平 = **截止**失真，原因是 **Q 点偏低**
- **填空30**：共射底部削平 = **饱和**失真，可**增大** $R_b$ 消除（$I_B = (V_{CC}-V_{BE})/R_b$，增大 $R_b$→$I_B$ 减小→Q 点下移）

### 5. 共集电极电路（射极跟随器）（填空 31、96）

- 又名**射极跟随器**（填空31）
- 电压放大倍数 **接近 1**（略小于 1，同相）（填空96）
- 输入电阻**大**，输出电阻**小**
- 常用于输入级、输出级、缓冲级（阻抗变换）

### 6. Q 点稳定（填空 32–33）

- **填空32**：影响 Q 点的主要因素是**温度**变化；稳定措施：引入**直流负反馈**
- **填空33**：分压式偏置电路中发射极电阻 $R_e$ 引入**直流电流串联负**反馈，稳定 $I_{CQ}$

### 7. 多级耦合方式（填空 34–37）

| 耦合方式 | 特点 | Q 点独立性 |
|----------|------|-----------|
| **阻容耦合** | 结构简单，低频电压放大常用 | 各级 Q 点**独立** |
| **直接耦合** | 集成运放必用，存在**零点漂移** | 前后级 Q 点**相互影响** |
| **变压器耦合** | 可实现阻抗匹配，体积大 | 各级独立 |

- **填空34–36**：低频放大常用**阻容**耦合（各级独立）；集成运放必须用**直接**耦合（需解决零漂）
- **填空37**：总电压放大倍数 = 各级之**积**；总通频带比单级**窄**

### 8. 频率响应（填空 38–40）

- **填空38**：通频带 $BW = f_H - f_L$
- **填空39**：$f_L$、$f_H$ 处增益下降 **3 dB**，附加相移约 **±45°**
- **填空40**：高频段增益下降原因 = **结**电容 + **分布**电容

### 9. MOS 管小信号模型（MOS 题 1–4）

- 跨导：$g_m = 2K_n(V_{GSQ} - V_{TN}) = 2\sqrt{K_n \cdot I_{DQ}}$
- 输出电阻：$r_{ds} = 1/(\lambda \cdot I_{DQ})$（题中常给 $\lambda = 0$，即 $r_{ds} \to \infty$）
- 共源电压增益：$A_v = -g_m(R_d \| R_L)$

### 10. BJT 小信号模型（BJT 题 1–4）

- $r_{be} = r_{bb'} + (1+\beta) \cdot \frac{26\,\text{mV}}{|I_{EQ}|\,\text{mA}}$
- 共射增益：$A_v = -\beta(R_c \| R_L) / r_{be}$

## 重点公式 / 结论速记

| 项目 | 公式/结论 |
|------|----------|
| 复合管 β | $\beta \approx \beta_1 \cdot \beta_2$ |
| 共集 $A_v$ | ≈ 1（同相） |
| 共射 $A_v$ | 相位**反**（180°） |
| 放大区偏置 | 发射结正偏、集电结反偏 |
| 多级增益 | 各级增益之**积** |
| 高频衰减来源 | 结电容 + 分布电容 |
| MOS 跨导 | $g_m = 2\sqrt{K_n \cdot I_{DQ}}$ |
| BJT $r_{be}$ | $r_{bb'} + (1+\beta) \cdot 26/I_{EQ}$ |
| 共源增益 | $A_v = -g_m(R_d \| R_L)$ |
| 共射增益 | $A_v = -\beta(R_c \| R_L)/r_{be}$ |

## 易错点提醒（对应题号）

| 题号 | 陷阱描述 | 正确结论 |
|------|----------|----------|
| **填空19** | BJT/FET 控制方式 | BJT=**电流**控制，FET=**电压**控制 |
| **填空20–21** | 放大区偏置 | 发射结**正偏**、集电结**反偏** |
| **填空24** | 输出特性三区 | 截止、**放大**、**饱和** |
| **填空28–30** | Q 点高低与失真类型 | 偏高→**饱和**（底部削平）；偏低→**截止**（顶部削平） |
| **填空30** | 消除饱和失真调电阻方向 | **增大** $R_b$（使 $I_B$ 减小，Q 下移） |
| **填空31** | 共集电极又名 | **射极跟随器** |
| **填空33** | 分压偏置 $R_e$ 引入何种反馈 | **直流电流串联负**反馈 |
| **填空34–36** | 低频用哪种耦合、集成运放用哪种 | 低频=**阻容**；集成=**直接**（有零漂） |
| **填空37** | 多级总增益和总通频带 | 增益=各级之**积**；通频带比单级**窄** |
| **MOS 题** | $\lambda = 0$ 时 $r_{ds}$ | 视为**开路**（无穷大） |
| **BJT 题4** | 开关 S 接 A/B/C 判断工作区 | 靠 $I_B$ 与 $I_{CS}$ 比较判截止/放大/饱和 |
