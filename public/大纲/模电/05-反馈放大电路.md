## 核心知识点

### 1. 反馈的分类（填空 48、50、52）

| 分类维度 | 类型 | 说明 |
|----------|------|------|
| 极性 | **正反馈** / **负反馈** | 增强 / 削弱净输入 |
| 信号性质 | **直流反馈** / **交流反馈** | 稳定 Q 点 / 改善动态性能 |
| 输出取样 | **电压反馈** / **电流反馈** | 取自输出电压 / 输出电流 |
| 输入叠加 | **串联反馈** / **并联反馈** | 电压形式 / 电流形式叠加 |

- **填空50**：稳定 Q 点用**直流**反馈；改善动态性能用**交流**反馈
- **填空52**：按输入端方式分**串联**反馈和**并联**反馈

### 2. 负反馈改善性能（填空 48）

负反馈可改善放大电路的性能：
- **展宽通频带**
- **减小非线性失真**（闭环内产生的失真）
- **稳定增益**（反馈深度越大越稳）
- **按需改变输入、输出电阻**

### 3. 四种组态对输入/输出电阻的影响（填空 49、51–53）

| 组态 | 输入电阻 | 输出电阻 | 稳定对象 | 教材章节 |
|------|----------|----------|----------|----------|
| **电压串联** | ↑ 增大 | ↓ 减小 | 输出电压 | §8.1 |
| **电压并联** | ↓ 减小 | ↓ 减小 | 输出电压 | §8.1 |
| **电流串联** | ↑ 增大 | ↑ 增大 | 输出电流 | §8.1 |
| **电流并联** | ↓ 减小 | ↑ 增大 | 输出电流 | §8.1 |

- **填空49**：稳定输出电压 + 提高输入电阻 → **电压串联负**反馈
- **填空51**：稳定输出电流 + 对电流信号源影响小 → **电流并联负**反馈
- **填空53**：电压负反馈稳定输出**电压**、减小输出**电阻**；电流负反馈稳定输出**电流**、增大输出**电阻**

**记忆口诀**：
- 电压负反馈→输出电阻↓、稳电压；电流负反馈→输出电阻↑、稳电流
- 串联负反馈→输入电阻↑；并联负反馈→输入电阻↓

### 4. 闭环增益与反馈深度（填空 49、54–56）

#### 4.1 闭环增益公式（填空 54）

$$A_F = \frac{A}{1+AF}$$

- $A$ 为开环增益，$F$ 为**反馈系数**

#### 4.2 反馈深度判据（填空 55）

- $|1+\dot{A}\dot{F}| > 1$ → **负反馈**
- $|1+\dot{A}\dot{F}| < 1$ → **正反馈**

#### 4.3 典型计算（填空 49）

$A = 1000$，$F = 0.099$：

$$A_f = \frac{A}{1+AF} = \frac{1000}{1 + 1000 \times 0.099} = \frac{1000}{100} = \boxed{10}$$

### 5. 自激振荡条件（填空 57–59）

#### 5.1 产生条件

负反馈在环路总附加相移达到 **180°** 时变为正反馈，可能产生**自激振荡**。

| 条件 | 公式 |
|------|------|
| 幅度条件 | $|AF| = 1$ |
| 相位条件 | 附加相移 $\Delta\varphi = (2n+1) \cdot 180°$（即 180° 的奇数倍） |

- **填空57**：反馈深度越大稳定性越好，但越容易在相移达 **180°** 时产生**自激振荡**；负反馈对频率特性的影响是**展宽**通频带
- **填空58**：波特图分析，当 $|AF| = 1$ 时若附加相移达 **180°**，电路将自激
- **填空59**：自激振荡的相移条件是**环路附加相移 $\varphi = (2n+1)\pi$**（180° 的奇数倍）

### 6. 反馈组态判断四步法（反馈题 1–4）

1. **找反馈支路**：从输出回连到输入回路的元件（常见为电阻）
2. **判极性**：用**瞬时极性法**，看反馈信号削弱还是增强净输入。削弱为负，增强为正
3. **判取样**（输出短路法）：输出短路后反馈消失 → **电压**反馈；仍在 → **电流**反馈
4. **判叠加**：反馈信号与输入接**同一节点** → **并联**；接**不同节点** → **串联**

## 重点结论 / 公式速记

| 项目 | 公式/结论 |
|------|----------|
| 闭环增益 | $A_f = A/(1+AF)$ |
| 反馈深度 | $1+AF$ |
| 负反馈判据 | $|1+AF| > 1$ |
| 自激幅度条件 | $|AF| = 1$ |
| 自激相位条件 | 附加相移 = 180° 奇数倍 |
| 串联反馈 | 输入电阻 ↑ |
| 并联反馈 | 输入电阻 ↓ |
| 电压反馈 | 输出电阻 ↓，稳电压 |
| 电流反馈 | 输出电阻 ↑，稳电流 |
| 负反馈对通频带 | **展宽** |
| 稳 Q 点用 | **直流**反馈 |
| 改善动态用 | **交流**反馈 |
