## 核心知识点

### 1. 锁存器 vs 触发器（判断 31，填空 33–34）

| 特性 | 锁存器（Latch） | 触发器（Flip-Flop） |
|------|-----------------|---------------------|
| 触发方式 | **电平敏感**（电平触发） | **边沿敏感**（边沿触发） |
| 状态变化 | 使能有效期间持续跟随输入 | 仅在时钟有效沿瞬间变化 |
| 透明态 | 使能有效时输出跟随输入 | — |
| 保持态 | 使能无效时输出保持 | — |

- **判断31陷阱**：触发器是**边沿触发**电路，不是电平触发电路。原题说"触发器属于电平触发"是**错的**
- **填空33**：锁存器属于**电平**敏感器件，触发器属于**边沿**敏感器件
- **填空34**：使能信号无效时输出保持，这种状态称为**保持态（锁存态）**

### 2. 基本 RS 触发器（判断 32–33）

#### 2.1 两种结构及其约束条件

| 结构 | 输入有效电平 | 约束条件 |
|------|-------------|----------|
| **或非门**构成 | 高电平有效 | S、R **不能同时为 1** |
| **与非门**构成 | 低电平有效（$\overline{S}$、$\overline{R}$） | $\overline{S}$、$\overline{R}$ **不能同时为 0** |

- **判断32陷阱**：用与非门构成的基本 RS 触发器，输入为低电平有效，约束条件是输入**不能同时为 0**（即 $\overline{S}=\overline{R}=0$ 禁止）。原题说"不能同时为 1"是**错的**（那是或非门构成的约束）
- **判断33陷阱**：原题"输入不能同时为 0"——若为与非门构成则**正确**；但题干未明确结构时默认为或非门，则约束应为"不能同时为 1"。答案判定此题为**错的**
- 特性方程：$Q^* = S + \overline{R} \cdot Q$，约束 $SR = 0$

### 3. 四种功能触发器特性方程（简答 3–4，必考）

#### 3.1 D 触发器（数据/延迟触发器）

$$Q^* = D$$

- 次态完全由 D 决定：D=0 置 0，D=1 置 1

#### 3.2 T 触发器（翻转触发器）— 填空 27，判断 29

$$Q^* = T \oplus Q = T \cdot \overline{Q} + \overline{T} \cdot Q$$

- **T = 0**：输出状态**保持**不变
- **T = 1**：每来一个时钟有效沿，输出状态**翻转**一次
- **判断29 ✓**：T 触发器在 T=1 时，每输入一个时钟脉冲状态翻转一次

#### 3.3 JK 触发器（通用触发器）— 填空 28

$$Q^* = J \cdot \overline{Q} + \overline{K} \cdot Q$$

| J | K | 功能 | 次态 |
|---|---|------|------|
| 0 | 0 | **保持** | Q |
| 0 | 1 | **置 0** | 0 |
| 1 | 0 | **置 1** | 1 |
| 1 | 1 | **翻转** | $\overline{Q}$ |

- **填空28**：当 J=K=1 时，每来一个时钟有效沿，输出**翻转**
- **JK 与 D 相比的优势**：D 只能置 0/置 1，JK 还能**保持**和**翻转**，功能最全，无需外加电路即可构成计数器

#### 3.4 SR 触发器

$$Q^* = S + \overline{R} \cdot Q, \quad \text{约束：} SR = 0$$

- S、R 不能同时为 1（否则次态不定）

### 4. JK 触发器的派生应用（简答 4）

- 两输入端**信号相反**时（令 $J = D, K = \overline{D}$）→ 构成 **D 触发器**
  - 代入：$Q^* = D\overline{Q} + D \cdot Q = D$
- 两输入端**信号相同**时（令 $J = K = T$）→ 构成 **T 触发器**
  - 代入：$Q^* = T\overline{Q} + \overline{T} \cdot Q = T \oplus Q$

### 5. 主从 JK 触发器一次翻转现象（填空 29）

- **一次翻转现象**：发生在时钟 **CP = 1（高电平）期间**
  - 在 CP=1 期间，若 J、K 发生变化，触发器只在 CP 下降沿前根据**初始状态**翻转一次，可能导致状态错误
- **克服方法**：改用**边沿触发器**（仅在时钟边沿瞬间采样 J、K 的值）

### 6. 异步置位/复位端（判断 30）

- 异步置位端（$\overline{S_D}$ / PRE）和异步复位端（$\overline{R_D}$ / CLR）
- **判断30 ✓**：异步置位端和异步复位端**不受时钟脉冲边沿触发控制**
- 优先级**高于**同步输入（J/K/D/T），用于上电初始化或强制清零
- 通常为**低电平有效**

### 7. 用 D 触发器实现 T 触发器（简答 10）

令 $D = T \oplus Q$，则 $Q^* = D = T \oplus Q$，即实现 T 触发器功能。
- 需要在 D 触发器输入端前加一个**异或门**，将 T 和 Q（反馈）异或后接到 D 端

## 重点公式 / 结论速记

| 触发器 | 特性方程 | 约束 | 教材章节 |
|--------|----------|------|----------|
| RS（基本） | $Q^* = S + \overline{R}Q$ | $SR = 0$ | §5.2 |
| D | $Q^* = D$ | 无 | §5.5 |
| T | $Q^* = T \oplus Q$ | 无 | §5.5 |
| JK | $Q^* = J\overline{Q} + \overline{K}Q$ | 无 | §5.5 |

- 锁存器 = **电平**敏感；触发器 = **边沿**敏感
- 与非门 RS：约束输入不能同时为 **0**
- 或非门 RS：约束输入不能同时为 **1**
- 异步置位/复位优先级最高，不受 CP 控制
- 主从 JK 一次翻转发生在 **CP=1 期间**，克服办法是**边沿**触发器

## 易错点提醒（对应题号）

| 题号 | 陷阱描述 | 正确结论 |
|------|----------|----------|
| **判断29** | T=1 时每来 CP 翻转一次 | ✓ **正确**（送分题） |
| **判断30** | 异步端不受时钟控制 | ✓ **正确** |
| **判断31** | 触发器属于电平触发 | 是**边沿**触发（**错**命题） |
| **判断32** | 与非门 RS 约束"不能同时为 1" | 约束是"不能同时为 **0**"（**错**命题） |
| **判断33** | 基本 RS 约束"不能同时为 0" | 默认或非门构成应为"不能同时为 **1**"（**错**命题） |
| **填空27** | T 触发器状态方程 | **$Q^* = T \oplus Q$**；T=0 保持，T=1 翻转 |
| **填空28** | J=K=1 时功能 | **翻转**（不是保持） |
| **填空29** | 主从 JK 一次翻转发生时间 + 克服 | 发生在 **CP=1 期间**；改用**边沿**触发器 |
| **简答3–4** | 特性方程默写 | D→$D$；T→$T \oplus Q$；JK→$J\overline{Q}+\overline{K}Q$；SR→$S+\overline{R}Q$ |
| **简答4** | JK 两输入相反/相同 | 相反→**D 触发器**；相同→**T 触发器** |
