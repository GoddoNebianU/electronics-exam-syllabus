## 核心知识点

### 1. 组合逻辑电路特点（判断 22–24）

- **特点**：任意时刻输出**仅取决于该时刻输入**，与电路原状态无关；**不存在反馈回路**
- **判断22陷阱**：组合电路虽无反馈，但因各信号路径传输延迟不同，**仍会出现**竞争冒险。原题说"不会出现"是**错的**
- **判断24陷阱**：译码器和数据选择器**都属于组合逻辑电路**。原题说"数据选择器属时序逻辑电路"是**错的**
- **判断26陷阱**：译码器是将**二进制代码**翻译为对应的输出信号，不是"翻译语言"。原题说法是**错的**

### 2. 竞争冒险（判断 22–23、27，填空 22）

#### 2.1 定义与成因

- **竞争**：信号通过不同路径到达终点的时间有先后（门传输延迟不同）
- **冒险**：竞争导致输出端出现不应有的尖峰脉冲（短暂错误）
- **判断23 ✓**：电路越简单、门级数越少，信号到达时间差越小，出现竞争冒险的可能性越小
- **判断27陷阱**：有竞争**不一定**产生冒险。只有**临界竞争**（到达时间差导致输出短暂跳变）才会出错，非临界竞争不影响输出。原题说"必定出现错误"是**错的**

#### 2.2 三种消除方法（填空 22）

1. **引入选通脉冲**：在信号稳定后取样（避开过渡期）
2. **输出端加滤波电容**：吸收尖峰脉冲（降低速度，简单有效）
3. **修改逻辑设计，添加冗余项**：增加冗余项消除冒险

### 3. 译码器（74LS138 / 74HC138）（填空 23–24，简答 23）

#### 3.1 基本结构

- **3 线-8 线译码器**：3 位地址输入（A₀A₁A₂），8 个输出（$Y_0$–$Y_7$）
- 输出**低电平有效**：每个输出对应一个最小项的非 $\overline{Y_i} = \overline{m_i}$

#### 3.2 实现逻辑函数（填空 23–24，简答 23）

- 将函数写成**最小项之和**形式 $F = \sum m(\ldots)$
- 译码器对应输出端接到**与非门**输入端
- **填空24**：`F(A,B,C)=∑m(1,3,5,7)` → 将 **Y₁、Y₃、Y₅、Y₇** 接到与非门输入端
  - $F = \overline{\overline{Y_1}·\overline{Y_3}·\overline{Y_5}·\overline{Y_7}} = m_1+m_3+m_5+m_7$
- **简答23（教材原题 4.4.8）**：用 74HC138 实现三变量四项乘积和的逻辑函数

### 4. 数据选择器（74151 / 74HC153）（填空 21、26，简答 12、18）

#### 4.1 八选一数据选择器 74151（填空 21）

- 共有 **3 个地址输入端**、**8 个数据输入端**
- 输出 $Y = \sum_{i=0}^{7} m_i \cdot D_i$（$m_i$ 为地址变量的最小项，$D_i$ 为对应数据输入）

#### 4.2 实现逻辑函数原理（填空 26）

- 地址输入端接**输入（选择）变量**
- 数据输入端接**常数 0 或 1**（对应于各最小项的函数值），或接剩余变量

### 5. 加法器（判断 25、28，填空 25）

#### 5.1 半加器与全加器

![](circuits/shu-full-adder.svg)
*图：一位全加器逻辑图——$S=A\oplus B\oplus C_{in}$，$C_{out}=AB+C_{in}(A\oplus B)$*

- **半加器**：仅考虑本位两个加数相加，不考虑低位进位
- **全加器**：既考虑低位来的进位输入 $C_{i-1}$，又考虑向高位的进位输出 $C_i$（判断28 ✓）
  - 本位和 $S_i = A_i \oplus B_i \oplus C_{i-1}$
  - 进位 $C_i = A_i B_i + (A_i \oplus B_i) C_{i-1}$

#### 5.2 超前进位加法器（判断 25，填空 25）

- **判断25陷阱**：超前进位加法器相对串行加法器的优点是**运算速度快**（进位并行产生），不是"电路更简单"（实际更复杂）。原题说法是**错的**
- **填空25**：四位超前进位加法器
  - 进位产生变量 $G_i = A_i \cdot B_i$
  - 进位传播变量 $P_i = A_i + B_i$（部分教材取 $P_i = A_i \oplus B_i$）
  - 进位链 $C_{i+1} = G_i + P_i \cdot C_i$

### 6. 组合电路设计题（简答 24–26，必考大题）

#### 6.1 设计四步法

1. **列真值表**：根据功能描述确定输入/输出变量，列出所有组合
2. **写逻辑表达式**：由真值表写最小项之和
3. **化简**：用卡诺图或公式法化到最简与或式
4. **画逻辑电路图**：用门电路实现

#### 6.2 简答24：空气净化多数表决

三台检测仪 A、B、C，两台以上超标 → 高电平启动。即**三变量多数表决电路**：

$$L = AB + BC + AC = AB + C(A+B)$$

#### 6.3 简答25：水箱液位检测

利用约束项（不可能的状态作为无关项 d），化简得：
- 红灯 $R = \overline{C}$
- 绿灯 $G = \overline{A}\,B$
- 黄灯 $Y = C(A + \overline{B})$

#### 6.4 简答26：举重裁判

甲（主）+ 至少一名副裁判（乙或丙）→ 合格：

$$\boxed{L = A(B + C) = AB + AC}$$
