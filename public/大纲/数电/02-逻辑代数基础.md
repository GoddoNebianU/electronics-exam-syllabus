## 核心知识点

### 1. 最小项与最大项（判断 9–13，填空 14）

#### 1.1 最小项（m）的性质

- **n 变量逻辑函数**的全部最小项共有 **$2^n$ 个**
  - **判断11陷阱**：原题说"n 个"是**错的**，正确为 $2^n$ 个
- 每个最小项由 **n 个因子**相乘构成，每个变量以原变量或反变量形式出现**恰好一次**
  - **判断10 ✓**：n 变量的一个最小项，其因子个数为 n 个
- 性质：任意一个最小项只有一组变量取值使其值为 1；全体最小项之和恒为 1

#### 1.2 最大项（M）的性质（填空 14）

- n 变量逻辑函数的最大项有 **$2^n$ 个**
- 每个最大项是 **n 个变量的或运算**（与最小项的"与运算"对应）
- 最小项与最大项互补：$m_i = \overline{M_i}$

#### 1.3 卡诺图化简规则（判断 9、12、13）

- **判断9 ✓**：合并的最小项个数必须是 **2 的整数次幂**（1、2、4、8、16…）
- **判断12 ✓**：**所有逻辑函数都可用卡诺图表示和化简**（变量多时手工不便，但理论上可行。答案判定为**正确**）
  > ⚠️ **修正**：旧版大纲此处表述模糊，现依据题库答案明确：判断12 答案为**正确**
- **判断13陷阱**：一个方格**可以**被多个卡诺圈包围（允许重叠），只需每个圈至少有一个**独占格**。原题说"只能被一个圈包围"是**错的**

### 2. 对偶规则与反演规则（填空 8–10，高频填空）

#### 2.1 对偶规则（求对偶式 Y′）

变换操作：`· ↔ +`（与↔或互换），`0 ↔ 1` 互换，**变量不变**，保持原运算优先级

| 原函数 Y | 对偶式 Y′ |
|----------|-----------|
| `A+B+C` | **`A·B·C`** |
| `AB+C` | **`(A+B)·C`** |
| `AB+C+D` | **`(A+B)·C·D`** |
| `A+BC` | **`A(B+C)`** |
| `A(B+C)` | **`A+BC`** |

#### 2.2 反演规则（求反函数 / 非函数 $\overline{Y}$）

变换操作：`· ↔ +`，`0 ↔ 1`，**每个变量取反**（原↔反互换），保持原运算优先级

| 原函数 Y | 反函数 $\overline{Y}$ |
|----------|----------------------|
| `AB+C` | **`$(\overline{A}+\overline{B})·\overline{C}$`** |
| `AB+C+D` | **`$(\overline{A}+\overline{B})·\overline{C}·\overline{D}$`** |
| `A+BC` | **`$\overline{A}·(\overline{B}+\overline{C})$`** |
| `A(B+C)` | **`$\overline{A}+\overline{B}·\overline{C}$`** |

> **对偶 vs 反演唯一区别**：对偶**不取反变量**，反演**取反变量**。

#### 2.3 用与非门实现逻辑函数（填空 8）

`Y = A+B+C` 用与非门实现：$Y = \overline{\overline{A}·\overline{B}·\overline{C}}$（先各变量取反求与非，再经一个与非门输出）

### 3. 无关项与卡诺图化简（填空 13，简答 5–7）

- 含有无关项的逻辑函数化简时，无关项在卡诺图中用 **×（或 d、φ）** 表示
- 无关项可任意视为 **0 或 1**，化简时灵活利用以获得最简表达式

### 4. 典型化简结论（填空 11–12）

- **填空11**：`F(A,B,C,D)=∑m(0,2,5,7,8,10,13,15)` 最简与或式 = **$\overline{B}\,\overline{D} + BD$**
  - 四角圈 m0,m2,m8,m10 → $\overline{B}\,\overline{D}$；m5,m7,m13,m15 → $BD$
- **填空12**：`Y=AB+AC+BC` 最简与或式 = **AB+AC+BC**（已是最简，不可再化简，此即"多数函数"）

## 重点公式 / 结论速记

| 项目 | 结论 |
|------|------|
| n 变量最小项个数 | **$2^n$**（不是 n） |
| n 变量最大项个数 | **$2^n$**，每个为 n 变量的**或** |
| 卡诺圈大小 | $2^k$（k=0,1,2,…），必须 2 的整数次幂 |
| 卡诺圈能否重叠 | **可以**（每圈至少有 1 个独占格） |
| 对偶规则 | `·↔+`，`0↔1`，**变量不变** |
| 反演规则 | `·↔+`，`0↔1`，**变量取反** |
| 无关项符号 | × 或 d，可视为 0 或 1 |

**常用公式补充**：
- 摩根定理：$\overline{A·B} = \overline{A}+\overline{B}$，$\overline{A+B} = \overline{A}·\overline{B}$
- 吸收律：$A+AB=A$，$A+\overline{A}B=A+B$
- 冗余项：$AB+\overline{A}C+BC=AB+\overline{A}C$

## 卡诺图化简详解（简答 5–7）

### 简答5（教材原题 2.4.3(4)）⭐

$L(A,B,C,D) = \sum m(0,4,6,13,14,15) + \sum d(1,2,3,5,7,9,10,11)$

利用无关项画三个 8 格大圈：

| 卡诺圈 | 覆盖范围 | 化简结果 |
|--------|----------|----------|
| 圈①（8格） | A=0 的两行（m0–m7） | **$\overline{A}$** |
| 圈②（8格） | C=1 的两列 | **$C$** |
| 圈③（8格） | D=1 的两列 | **$D$** |

$$\boxed{L = \overline{A} + C + D}$$

### 简答6（变式）

$L(A,B,C,D) = \sum m(0,2,4,6,9,13) + \sum d(1,3,5,7,11,15)$

两个 8 格大圈：$\overline{A}$（A=0 两行）+ $D$（D=1 两列）

$$\boxed{L = \overline{A} + D}$$

### 简答7（变式）

$L(A,B,C,D) = \sum m(0,2,3,4,6,11,12) + \sum d(8,9,10,13,14,15)$

8 格圈 $\overline{D}$（D=0 两列）+ 4 格圈 $\overline{B}C$（B=0, C=1）

$$\boxed{L = \overline{D} + \overline{B}\,C}$$

## 易错点提醒（对应题号）

| 题号 | 陷阱描述 | 正确结论 |
|------|----------|----------|
| **判断11** | "n 变量全部最小项共 n 个" | 共 **$2^n$** 个（**错**命题） |
| **判断12** | "所有逻辑函数都可用卡诺图化简" | **正确** ✓（理论可行，手工仅变量多时不便） |
| **判断13** | "每方格只能被一个卡诺圈包围" | 方格**可重叠**被多圈包围（**错**命题） |
| **填空8–10** | 对偶式和反函数混淆 | 对偶：变量**不变**；反演：变量**取反** |
| **填空11** | ∑m(0,2,5,7,8,10,13,15) 化简 | 最简与或式 = **$\overline{B}\,\overline{D} + BD$** |
| **填空12** | Y=AB+AC+BC 能否继续化简 | **已是最简**（三项均为本质素项） |
| **填空14** | n 变量最大项个数和运算类型 | **$2^n$** 个，每个为 n 变量的**或**运算 |
| **简答5–7** | 不会利用无关项画大圈 | 应合理把 d 当 1，尽量画 8 格大圈 |
