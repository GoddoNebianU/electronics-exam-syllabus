# 第6章 信号的运算与处理

> **题型分布**：填空 60–72、电压比较器题 1–2、综合设计题 1–2
> **重要度**：★★★（运放应用是模电主战场，与综合设计章紧密关联）

## 核心知识点

### 1. 理想运放两大核心特性（填空60、100）
- **虚短**：`U+ = U−`（两输入端电位相等）。
- **虚断**：`I+ = I− = 0`（两输入端电流为零）。
- 虚短成立的前提：电路引入**深度负反馈**且工作在**线性区**。
- 比较器、振荡器等开环或正反馈电路**不能用虚短**（填空70）。

### 2. 比例运算电路（填空61、62）
| 电路 | 输入端 | 电压放大倍数 | 输入电阻 |
|------|--------|--------------|----------|
| 反相比例 | 接**反相**端（同相端接地） | `Av = −Rf/R1` | 等于 R1（较小） |
| 同相比例 | 接**同相**端 | `Av = 1 + Rf/R1 > 1` | 很大 |

- 同相比例电路电压放大倍数恒**大于 1**（填空61）。
- 反相比例输入电阻 = `R1`（受限于 R1，不能太大）。
- **电压跟随器**：同相比例特例，`Rf = 0`，`Av = 1`，输入电阻极大、输出电阻极小。

### 3. 求和与加减运算（填空64、综合设计1、2）
- **反相求和**：多路输入分别经电阻接运放**反相**输入端，输出与各输入电压代数和成**反相比例**关系（填空64）。
  - `vo = −(Rf/R1·v1 + Rf/R2·v2 + ...)`，各路权重由 `Rf/Ri` 决定。
- **同相求和**：信号接同相端。
- **加减运算**：反相端接减项，同相端接加项（综合设计1 中系数 a>0、abc<0 即用此）。

### 4. 积分与微分运算（填空63、65）
| 电路 | 关键元件位置 | 输出含义 |
|------|--------------|----------|
| 积分 | **电容在反馈支路** | 输出是输入的**积分**（`vo = −1/RC ∫vi dt`） |
| 微分 | **电容在输入支路** | 输出是输入的**微分**（`vo = −RC·dvi/dt`） |

- 积分电路能把方波变三角波、把正弦波移相 +90°（综合设计1）。
- 微分电路能把方波转**尖顶波**（综合设计1），对阶跃边沿敏感。

### 5. 有源滤波器（填空66、67、68、69）
- **滤波器**：具有**选频**功能的电子装置，让某些频率通过、其他频率抑制。
- **有源滤波器** = **运算放大器** + **RC 网络**（填空66、67）。
- 按通带分类：**低通、高通、带通、带阻**四种（填空69）。
- 抑制 50 Hz 工频干扰 → 选用**带阻**滤波器（陷波器，填空68）。
- 有源滤波相比无源滤波：带负载能力强、可提供增益、无需电感。

### 6. 电压比较器（填空70、71、72）
- 比较器工作在**非线性区**（开环或正反馈），输出只有**高电平、低电平**两种状态（填空70、72）。
- **滞回比较器**：具有**回差**特性（迟滞传输特性），可提高**抗干扰**能力（填空71）。
- 比较器是模拟到数字的接口电路，常用于电平检测、波形整形。

## 重点结论 / 公式速记

| 项目 | 公式/结论 |
|------|----------|
| 反相比例 | `Av = −Rf/R1` |
| 同相比例 | `Av = 1 + Rf/R1` |
| 电压跟随器 | `Av = 1` |
| 反相求和 | `vo = −Σ(Rf/Ri · vi)` |
| 积分 | `vo = −(1/RC)∫vi dt` |
| 微分 | `vo = −RC · dvi/dt` |
| 比较器工作区 | 非线性区 |
| 滞回比较器特点 | 回差特性，抗干扰 |
| 有源滤波器组成 | 运放 + RC |
| 滤波器四类 | 低通、高通、带通、带阻 |

## 电压比较器题解题要点（比较器题1、2）
1. 求门限电压（阈值）：令 `U+ = U−`，解出对应的 `vi` 值。
2. 滞回比较器有**两个**门限：上行门限 `UTH+` 和下行门限 `UTH−`，回差 `ΔUT = UTH+ − UTH−`。
3. 画传输特性时标出转折点坐标和滞回方向（顺时针/逆时针）。
4. 题中稳压管双向限幅 ±6 V（题2）或 `VZ = 9 V`（题1），输出被钳在该电压。
5. 单限比较器只有一个门限，传输特性是单段跳变。

## 易错点提醒
- **虚短成立必须满足深度负反馈**；比较器没有负反馈（开环或正反馈），**不能用虚短**。
- 反相比例电路输入电阻等于 `R1`（不是 `Rf`，也不是运放本身），设计时要权衡增益与输入电阻。
- 同相比例电路存在**共模输入**，对运放共模抑制比有要求；反相比例无共模输入。
- 积分电路的电容在**反馈**支路；微分电路的电容在**输入**支路，二者位置**相反**，最易混淆。
- 滤波器选型：去除高频 → 低通；抑制 50 Hz → 带阻（不是低通，因为 50 Hz 不算高频）。
- 电压比较器题画传输特性必须**标转折点坐标**，否则扣分。
- 滞回比较器"回差"是抗干扰的核心，回差越大抗干扰越强但灵敏度越低。
- 同相比例 `Av = 1 + Rf/R1` 恒大于 1，**不能**实现衰减；要实现 `Av < 1` 需用反相比例或分压。
