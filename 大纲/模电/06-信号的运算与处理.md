## 核心知识点

### 1. 理想运放两大核心特性（填空 60、100）

| 特性 | 含义 | 前提条件 |
|------|------|----------|
| **虚短** | $U_+ = U_-$（两输入端电位相等） | 引入**深度负反馈**且工作在**线性区** |
| **虚断** | $I_+ = I_- = 0$（两输入端电流为零） | 同上 |

- **填空60**：理想运放核心特性为虚短和**虚断**，虚短前提是引入**深度**负反馈
- **填空100**：理想运放线性区两特性：**虚短、虚断**
- 比较器、振荡器等**开环或正反馈**电路**不能用虚短**（填空70）

### 2. 比例运算电路（填空 61–62）

![](circuits/mo-inverting-amp.svg)
*图：反相比例运算电路——信号经 $R_1$ 接反相端，$A_v=-R_f/R_1$*

![](circuits/mo-noninverting-amp.svg)
*图：同相比例运算电路——信号接同相端，$A_v=1+R_f/R_1>1$（$R_f=0$ 即电压跟随器）*

| 电路 | 输入端 | 电压放大倍数 | 输入电阻 |
|------|--------|--------------|----------|
| **反相比例** | 接**反相**端（同相端接地） | $A_v = -R_f/R_1$ | 等于 $R_1$（较小） |
| **同相比例** | 接**同相**端 | $A_v = 1 + R_f/R_1 > 1$ | 很大 |

- **填空61**：反相比例输入接**反相**端；同相比例放大倍数恒**大于** 1
- **填空62**：同相比例输入接**同相**端，输入电阻**很大**
- **电压跟随器**：同相比例特例（$R_f = 0$），$A_v = 1$，输入电阻极大、输出电阻极小

### 3. 求和与加减运算（填空 64）

- **反相求和**：多个输入信号分别经电阻接运放**反相**端，输出与各输入电压代数和成**正比**

$$v_o = -\left(\frac{R_f}{R_1}v_{i1} + \frac{R_f}{R_2}v_{i2} + \ldots\right)$$

- **加减运算**：反相端接减项，同相端接加项（综合设计题1 中 $a>0, abc<0$ 即用此）

### 4. 积分与微分运算（填空 63、65）

![](circuits/mo-integrator.svg)
*图：积分运算电路——电容在反馈支路，$v_o=-(1/RC)\int v_i\,dt$（方波→三角波）*

| 电路 | 关键元件位置 | 输出含义 | 公式 |
|------|--------------|----------|------|
| **积分** | **电容在反馈支路** | 输出是输入的**积分** | $v_o = -\frac{1}{RC}\int v_i\,dt$ |
| **微分** | **电容在输入支路** | 输出是输入的**微分** | $v_o = -RC\frac{dv_i}{dt}$ |

- **填空63**：积分电路反馈支路元件是**电容**，输出是输入的**积分**
- **填空65**：微分电路输入支路元件是**电容**，输出是输入的**微分**
- 积分电路能把方波变三角波、把正弦波移相 +90°（综合设计题1）
- 微分电路能把方波转**尖顶波**（综合设计题1）

### 5. 有源滤波器（填空 66–69）

#### 5.1 基本概念

- **滤波器**：具有**选频（频率选择）**功能的电子装置
- **有源滤波器** = **运算放大器（有源器件）** + **RC 网络（阻容网络）**（填空66、67）

#### 5.2 分类（填空 69）

按通带频率范围分四种：**低通、高通、带通、带阻**

#### 5.3 典型选型（填空 68）

- 抑制 50Hz 交流电源干扰 → 选用**带阻（陷波）**滤波器

### 6. 电压比较器（填空 70–72，比较器题 1–2）

#### 6.1 基本特性

- 比较器工作在**开环（非线性/饱和）**状态（填空70）
- 输出只有**高电平、低电平**两种状态（填空70、72）
- 集成运放工作在**非线性区**（填空72）
- 比较器是模拟到数字的接口电路

#### 6.2 滞回比较器（填空 71）

- 具有**回差（滞回）**特性（迟滞传输特性）
- 可提高电路的**抗干扰**能力
- 有**两个**门限电压：上行门限 $U_{TH+}$ 和下行门限 $U_{TH-}$
- 回差 $\Delta U_T = U_{TH+} - U_{TH-}$

### 7. 电压比较器题解题要点（比较器题 1–2）

1. **求门限电压（阈值）**：令 $U_+ = U_-$，解出对应的 $v_i$ 值
2. **画传输特性**：标出转折点坐标和滞回方向
3. 题1：$V_Z = 9\,\text{V}$（教材原题 10.8.4）
4. 题2：稳压管双向限幅 ±6V（教材原题 10.8.6）

## 重点公式 / 结论速记

| 项目 | 公式/结论 |
|------|----------|
| 反相比例 | $A_v = -R_f/R_1$ |
| 同相比例 | $A_v = 1 + R_f/R_1$（恒 > 1） |
| 电压跟随器 | $A_v = 1$ |
| 反相求和 | $v_o = -\sum(R_f/R_i \cdot v_i)$ |
| 积分 | $v_o = -(1/RC)\int v_i\,dt$ |
| 微分 | $v_o = -RC \cdot dv_i/dt$ |
| 比较器工作区 | 非线性区（开环） |
| 滞回比较器 | 回差特性，抗干扰 |
| 有源滤波器组成 | 运放 + RC |
| 滤波器四类 | 低通、高通、带通、带阻 |
| 虚短前提 | 深度负反馈 + 线性区 |
