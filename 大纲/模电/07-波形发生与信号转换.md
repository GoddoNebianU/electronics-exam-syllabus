# 第7章 波形发生与信号转换

> **题型分布**：填空 73–78、正弦波振荡电路题 1–4
> **重要度**：★★（题量较少，但文氏电桥是必考含图题）

## 核心知识点

### 1. 正弦波振荡的平衡条件（填空73、74）
- **幅度平衡条件**：`|AF| = 1`
- **相位平衡条件**：环路总相移 `Σφ = 2nπ`（n 为整数）
- **起振条件**：幅度 `|AF| > 1`，相位 `Σφ = 2nπ`
- 从起振到稳幅：随幅度增大，放大器件进入非线性区，增益 `A` 自动下降，直到 `|AF| = 1` 稳定振荡。
- 振荡电路组成：放大电路 + 选频网络 + 正反馈网络 + 稳幅环节。

### 2. RC 正弦波振荡电路（填空75、振荡题1–4）
- 适用产生**低频**信号（频率范围一般 1 Hz ~ 1 MHz）。
- 选频网络由 **R 和 C** 组成（文氏电桥/串并联 RC 网络）。
- 文氏电桥振荡频率：

$$f_0 = \frac{1}{2\pi RC}$$

- 文氏电桥在 `f0` 处：反馈系数 `F = 1/3`，相移为 0（满足相位条件）。
- 为满足起振条件 `|AF| > 1`，要求 `A > 3`，即同相放大倍数 `1 + Rf/R1 > 3`，故 `Rf > 2 R1`。
- 振荡题2 中 `Rp + R2` 即同相比例的反馈电阻，应大于 `2 × 输入电阻` 才能起振。

### 3. LC 正弦波振荡电路（对比）
- 选频网络由 L 和 C 组成，适用于**高频**信号（> 1 MHz）。
- 常见形式：变压器反馈式、电感三点式（哈特莱）、电容三点式（考毕兹）。
- 题库以 RC 文氏电桥为主，LC 仅作对比了解。

### 4. 方波发生器（填空76、78）
- 由 **滞回比较器** + **RC 充放电回路（积分）** 组成。
- 工作原理：输出经 RC 反馈到反相端，电容电压在两个门限之间反复充放电，输出方波。
- 输出电压幅度由**稳压管限幅值**（或电源电压）决定（填空78）。
- 振荡频率由 **RC 时间常数** 和**滞回比较器阈值**共同决定（填空78）。
- 频率公式：`f ≈ 1/(2RC·ln(1+2R1/R2))`（取决于比较器电阻比）。

### 5. 三角波发生器（填空77）
- 由 **方波发生器** + **积分器** 两部分级联构成。
- 方波经积分得到三角波。
- 若要锯齿波，让积分器充放电时间常数不等即可。

### 6. 文氏电桥题常见问题（振荡题1–4）
- **题1**：**热敏电阻 Rt** 用于稳幅。要保证稳幅时负反馈自动增强，Rt 应有**负温度系数**（电流增大 → 温度升高 → 阻值减小 → 负反馈增强 → 增益下降到 3，达到 `|AF| = 1`）。求稳幅时 `It`、`Rt` 值与输出 `VO`。
- **题2**：标运放同相端/反相端，使得文氏电桥接**同相端**满足正反馈（相位条件），分压电阻接**反相端**构成负反馈稳幅。求 `Rp + R2` 起振下限（即 `> 2 R1`）。
- **题3**：选频网络由 RC 串并联元件组成；负反馈网络由 Rt、R1 组成；稳定工作时 `Rt = 2 R1`；热敏电阻为负温度系数。
- **题4**：电路不振荡常因为反馈极性错误（同相反相接反）或 `A ≤ 3`（电阻比不满足），需改正接线/调电阻。

## 重点结论 / 公式速记

| 项目 | 公式/结论 |
|------|----------|
| 振荡幅度平衡 | `|AF| = 1` |
| 振荡相位平衡 | `Σφ = 2nπ` |
| 起振幅度 | `|AF| > 1` |
| 文氏电桥频率 | `f0 = 1/(2πRC)` |
| 文氏电桥反馈系数 | `F = 1/3`（在 f0 处） |
| 起振增益要求 | `A > 3`，即 `Rf > 2 R1` |
| 方波发生器组成 | 滞回比较器 + RC |
| 三角波发生器 | 方波 + 积分器 |
| 稳幅热敏电阻 | 负温度系数 |
| RC 振荡适用 | 低频 |
| LC 振荡适用 | 高频 |

## 易错点提醒
- 起振条件是 `|AF| > 1`（**严格大于**），平衡条件才是 `|AF| = 1`，二者不可混淆。
- 文氏电桥 RC 网络必须接**同相**输入端（构成正反馈），分压电阻网络接**反相**输入端（负反馈稳幅），位置颠倒则不振荡。
- "选频网络由 R 和 C 组成"——勿写成"由 LC 组成"（LC 是另一类振荡器，适合高频）。
- 稳幅用热敏电阻时，应选**负温度系数**才能自动稳幅（电流大→升温→阻值降→负反馈加深→降增益）。
- 方波发生器频率**不是**仅由 RC 决定，还与滞回比较器门限（即 `R1/R2` 比值）有关。
- 文氏电桥振荡题画运放接线时，**同相端**接 RC 串并联选频网络中点，**反相端**接分压电阻。
- 起振电阻条件 `Rf > 2 R1`（不是 `Rf > R1`），其中 `Rf` 是负反馈总电阻。
- 题3 中"稳定工作"意味着 `|AF| = 1`，此时 `A = 3`，由 `1 + Rt/R1 = 3` 得 `Rt = 2 R1`。
- 三角波发生器是把方波送入**积分器**，不是微分器；积分对方波产生线性斜坡。
- 振荡电路必须有**选频网络**决定振荡频率，**正反馈网络**满足相位条件，**稳幅环节**保证 `|AF|=1`。

## 典型计算示例
- 文氏电桥 `R = 16 kΩ, C = 0.01 μF`：`f0 = 1/(2π·16k·0.01μ) ≈ 995 Hz ≈ 1 kHz`。
- 起振条件 `Rf > 2 R1`：若 `R1 = 10 kΩ`，则负反馈总电阻 `Rf`（含热敏电阻）必须 > 20 kΩ 才能起振。
- 方波发生器频率 `f ≈ 1/(2RC·ln(1+2R1/R2))`，与门限比值 `R1/R2` 相关。
