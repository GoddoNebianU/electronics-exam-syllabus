## 核心知识点

### 1. 放大电路的四种类型<sup>填空1</sup>

按输入量、输出量是电压还是电流划分：

| 类型 | 输入量 | 输出量 | 增益量纲 | 典型应用 |
|------|--------|--------|----------|----------|
| **电压放大** | 电压 | 电压 | 无量纲（V/V） | 音频前置放大 |
| **电流放大** | 电流 | 电流 | 无量纲（A/A） | 电流驱动 |
| **互阻放大** | 电流 | 电压 | Ω | 光电接收 |
| **互导放大** | 电压 | 电流 | S | 电压转电流 |

四种类型本质上是信号源与负载的不同组合方式，决定了放大电路的输入电阻、输出电阻要求。

### 2. 主要性能指标<sup>填空2</sup>

衡量放大电路的五大标准：**增益（放大倍数）、输入电阻、输出电阻、通频带、非线性失真**。

| 指标 | 物理意义 |
|------|----------|
| 增益 $A$ | 放大能力 |
| 输入电阻 $R_i$ | 对信号源的要求 |
| 输出电阻 $R_o$ | 带负载能力 |
| 通频带 $BW$ | 频率适应范围 |
| 非线性失真 | 信号保真度 |

### 3. 输入电阻与输出电阻的物理意义<sup>填空3</sup>

- **输入电阻 $R_i$** 越**大**，从信号源汲取的电流越小，对信号源要求越低
- **输出电阻 $R_o$** 越**小**，带负载能力越强，负载变化时输出电压越稳定
- 工程口诀：电压放大要"高入低出"（$R_i$ 大、$R_o$ 小）

### 4. 通频带定义<sup>填空4、38</sup>

- 通频带 $BW = f_H - f_L$：**上限频率** $f_H$ 与**下限频率** $f_L$ 之间的频率范围
- 在 $f_L$ 与 $f_H$ 处，增益相对中频下降 **3 dB**，此时附加相移约为 **±45°**<sup>填空39</sup>
- 高频段增益下降原因：**结电容、分布电容**<sup>填空40</sup>
- 低频段增益下降原因：耦合电容、旁路电容

### 5. 输出电阻的计算<sup>计算题1–3</sup>⭐ 必考

#### 5.1 负载开路法（戴维南等效）

设负载开路输出电压为 $V_{o\text{开}}$，接负载 $R_L$ 时输出为 $V_o$：

$$R_o = \left(\frac{V_{o\text{开}}}{V_o} - 1\right) R_L$$

#### 5.2 计算题2（教材原题 1.5.3）

$V_{o\text{开}} = 1.1\,\text{V}$，$V_o = 1\,\text{V}$，$R_L = 1\,\text{k}\Omega$：

$$R_o = \left(\frac{1.1}{1} - 1\right) \times 1000 = \boxed{100\,\Omega}$$

#### 5.3 计算题3（教材原题 1.5.2）

"减少 20%"即 $V_o = 0.8 \times V_{o\text{开}}$，$R_L = 1\,\text{k}\Omega$：

$$R_o = \left(\frac{1}{0.8} - 1\right) \times 1000 = \boxed{250\,\Omega}$$

### 6. 阻抗匹配与戴维南等效<sup>计算题1</sup>（教材原题 1.4.2）⭐

**已知**：拾音头 $R_s = 1\,\text{M}\Omega$、$V_s = 1\,\text{V}$，扬声器 $R_L = 10\,\Omega$。放大电路 $R_i = 1\,\text{M}\Omega$，$R_o = 10\,\Omega$，开路增益 $A_v = 1$。

**(1) 直接相接**（无放大电路）：

$$V_L = V_s \cdot \frac{R_L}{R_s + R_L} = 1 \times \frac{10}{10^6 + 10} \approx 10\,\mu\text{V}$$

**(2) 接入放大电路后**：

- 输入侧分压：$V_{入} = 1 \times \frac{10^6}{10^6 + 10^6} = 0.5\,\text{V}$
- 开路输出：$V_{o\text{开}} = A_v \times V_{入} = 0.5\,\text{V}$
- 输出侧分压：$V_L = 0.5 \times \frac{10}{10 + 10} = \boxed{0.25\,\text{V}}$

**结论**：放大电路实现了**阻抗变换（缓冲隔离）**，即使电压增益为 1，扬声器电压从 10 μV 提升到 0.25 V，提高 **25000 倍**。

### 7. 四种放大电路的阻抗要求对比<sup>填空1、3</sup>

| 类型 | 输入电阻要求 | 输出电阻要求 |
|------|--------------|--------------|
| 电压放大 | $R_i$ **大** | $R_o$ **小** |
| 电流放大 | $R_i$ **小** | $R_o$ **大** |
| 互阻放大 | $R_i$ **小** | $R_o$ **小** |
| 互导放大 | $R_i$ **大** | $R_o$ **大** |

## 重点公式 / 结论速记

| 项目 | 公式/结论 |
|------|----------|
| 通频带 | $BW = f_H - f_L$ |
| $f_L$/$f_H$ 处增益衰减 | −3 dB（幅度降到中频 0.707 倍） |
| $f_L$/$f_H$ 处附加相移 | ±45° |
| 输出电阻（两次测量法） | $R_o = (V_{o\text{开}}/V_o - 1) \cdot R_L$ |
| 阻抗匹配目标 | $R_i \gg R_s$，$R_o \ll R_L$ |
| 多级总增益 | 各级增益之**积** |
| 多级总通频带 | 比单级**窄** |

## 易错点提醒（对应题号）

| 题号 | 陷阱描述 | 正确结论 |
|------|----------|----------|
| **填空1** | 四种放大类型 | 电压、电流、**互阻**、**互导** |
| **填空3** | 输入电阻大→汲取电流？输出电阻小→带载强？ | 输入电阻越**大**汲取越小；输出电阻越**小**带载越强 |
| **填空4** | 通频带范围 | **下限**频率与**上限**频率之间 |
| **计算题1** | 增益为 1 为何还能提升负载电压 | 阻抗变换：高 $R_i$ 减少输入损失，低 $R_o$ 减少输出损失 |
| **计算题2** | "减少 20%"的含义 | 带载输出是开路的 **80%**（$V_o/V_{o\text{开}} = 0.8$），不是减少到 20% |
| **计算题3** | 同上 | $R_o = (1/0.8 - 1) \times 1000 = 250\,\Omega$ |
