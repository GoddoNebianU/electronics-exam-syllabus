# 第5章 反馈放大电路

> **题型分布**：填空 48–59、反馈判断题 1–4、差分综合题（级间反馈）、频率响应综合题
> **重要度**：★★★（概念题密集，判断组态是必考题型）

## 核心知识点

### 1. 反馈的分类（填空48、50、52）
- **正反馈 / 负反馈**：反馈信号增强 / 削弱净输入。
- **直流反馈 / 交流反馈**：稳定 Q 点用直流反馈，改善动态性能用交流反馈。
- **电压反馈 / 电流反馈**：反馈取样自输出**电压**（与负载并联取样）还是输出**电流**（与负载串联取样）。
- **串联反馈 / 并联反馈**：反馈信号在输入端以**电压**形式（串联）还是**电流**形式（并联）叠加。

### 2. 负反馈改善性能（填空48）
- **展宽通频带**。
- **减小非线性失真**（闭环内产生的失真）。
- **稳定增益**（反馈深度越大越稳）。
- **按需改变输入、输出电阻**。

### 3. 四种组态对输入/输出电阻的影响（填空49、51、52、53）
| 组态 | 输入电阻 | 输出电阻 | 稳定对象 |
|------|----------|----------|----------|
| 电压串联负反馈 | ↑ 增大 | ↓ 减小 | 输出电压 |
| 电压并联负反馈 | ↓ 减小 | ↓ 减小 | 输出电压 |
| 电流串联负反馈 | ↑ 增大 | ↑ 增大 | 输出电流 |
| 电流并联负反馈 | ↓ 减小 | ↑ 增大 | 输出电流 |

- **电压负反馈**稳定输出**电压**、减小输出**电阻**。
- **电流负反馈**稳定输出**电流**、增大输出**电阻**。
- **串联负反馈**提高输入电阻；**并联负反馈**降低输入电阻。

### 4. 闭环增益与反馈深度（填空49、54、55、56）
- 闭环增益公式：

$$A_F = \frac{A}{1+AF}$$

- `A` 为开环增益，`F` 为反馈系数。
- 反馈深度 `|1+AF|`：
  - `|1+AF| > 1` → **负反馈**。
  - `|1+AF| < 1` → **正反馈**。
- 填空49典型计算：`A=1000, F=0.099` → `AF = 99` → `Af = 1000/100 = 10`。
- 反馈深度定义：加入反馈后达到相同输出所需输入信号之比。

### 5. 自激振荡条件（填空57、58、59）
- 当环路总附加相移达到 **180°** 时，负反馈变成正反馈。
- 自激振荡条件（同时满足）：
  - **幅度条件**：`|AF| = 1`
  - **相位条件**：附加相移 `Δφ = (2n+1)·180°`
- 反馈深度越大稳定性越好，但越容易在相移达 180° 时产生**自激振荡**。
- 常用**波特图**判稳：`|AF| = 1` 时若附加相移已 ≥ 180° 则自激。

## 重点结论 / 公式速记

| 项目 | 公式/结论 |
|------|----------|
| 闭环增益 | `Af = A/(1+AF)` |
| 反馈深度 | `1+AF` |
| 负反馈判据 | `1+AF > 1` |
| 自激幅度条件 | `|AF| = 1` |
| 自激相位条件 | 附加相移 = 180°（即 `(2n+1)π`） |
| 串联负反馈影响 | 输入电阻 ↑ |
| 电压负反馈影响 | 输出电阻 ↓，稳定电压 |
| 通频带变化 | 负反馈展宽通频带 |

## 反馈组态判断四步法（反馈题1–4）
1. **找反馈支路**：从输出回连到输入回路的元件（电阻常见）。
2. **判极性**：用瞬时极性法，看反馈信号是削弱还是增强净输入。削弱为负，增强为正。
3. **判取样**：输出短路法。输出短路后反馈消失 → **电压**反馈；仍在 → **电流**反馈。
4. **判叠加**：反馈信号与输入接在同一节点 → **并联**；接在不同节点（不同电极）→ **串联**。

## 易错点提醒
- "稳定输出电压 + 提高输入电阻" = **电压串联**负反馈（填空49、综合98）。
- "稳定输出电流 + 对电流信号源影响小"对应 **电流并联**负反馈（填空51，因电流源内阻大需并联低输入电阻）。
- 闭环增益公式 `Af = A/(1+AF)` 中 `A` 与 `F` **必须是同一种**（同为电压、同为电流等）。
- 填空55的符号 `|1+ȦḞ|` 带圆圈表示**复数**，要看模与相位综合判断，不要只看模值。
- 自激振荡判稳时，附加相移达到 **180°**（不是 90°）才使负反馈变成正反馈。
- 多级放大电路级间反馈通常是**主要反馈**，单级局部反馈（如发射极电阻 Re 的电流串联负反馈）次之，判断时优先看级间。
- 稳定静态工作点用**直流负反馈**，改善动态性能（增益、频带、失真）用**交流负反馈**（填空50）。
- 反馈深度 `|1+AF|` 越大，闭环增益越稳定，但增益绝对值**减小**（用增益换稳定）。
- 串联反馈适用于低内阻信号源（电压源），并联反馈适用于高内阻信号源（电流源）。
