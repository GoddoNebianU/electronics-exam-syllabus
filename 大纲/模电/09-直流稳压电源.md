## 核心知识点

### 1. 直流稳压电源四大组成部分（填空 87–89、95）

按顺序依次为：

| 序号 | 环节 | 功能 | 关键元件 |
|------|------|------|----------|
| 1 | **电源变压器** | 把 220V 交流市电**降压**到所需交流电压 | 铁芯线圈 |
| 2 | **整流电路** | 把交流转成**脉动直流**（统一电流方向） | 二极管（桥式） |
| 3 | **滤波电路** | 把脉动直流**平滑**为较平直直流 | 电容/电感 |
| 4 | **稳压电路** | 保证输出电压**恒定**（不随电网/负载波动） | 调整管/稳压管 |

- **填空87**：线性稳压电源（串联型）四部分 = **电源变压器、整流电路、滤波电路、稳压电路**
- **填空88**：变压器承担**电压转换**；整流实现**电流方向统一**；滤波使**波形平滑**；稳压确保**电压恒定**
- **填空89**：四大组成依次为电源变压器、**整流电路**、滤波电路、**稳压电路**
- **综合填空95**：变压器作用=**降压**（交流电网电压→低压交流）；整流作用=**变直流**（交流→脉动直流）

### 2. 滤波电路（填空 90）

- 常用三类：**电容滤波、电感滤波、LC（π型/复式）滤波**

| 类型 | 连接方式 | 特点 |
|------|----------|------|
| **电容滤波** | 与负载**并联** | 空载 $U_o \approx \sqrt{2}U_2$；简单但带载差 |
| **电感滤波** | 与负载**串联** | 适用于大电流负载；体积大 |
| **LC 滤波** | 电感+电容组合 | 纹波更小，兼顾两者 |

### 3. 串联反馈式稳压电路（填空 92）

![](circuits/mo-series-regulator.svg)
*图：串联型稳压电源——取样 + 基准 + 比较放大 + 调整管（本质为电压串联负反馈闭环）*

四大组成部分：

| 组成 | 功能 |
|------|------|
| **取样电路** | 从输出端分压取样（电阻分压网络） |
| **基准电压** | 由稳压管提供（恒定参考） |
| **比较放大器** | 取样电压与基准电压比较，差值放大 |
| **调整管** | 根据放大结果控制导通程度，调节输出 |

- 信号流向：**取样电路**的输出与**基准电压**在**比较放大器**的输入端汇合比较，差值放大后控制**调整管**（填空92）
- 本质是**电压串联负反馈**闭环系统

### 4. 硅稳压管稳压电路（填空 93）

- **限流电阻**作用：**限制电流**（保护稳压管并配合稳压），同时与负载**串联**连接
- 稳压管与负载**并联**（反向），限流电阻**串联**在输入与负载之间

限流电阻计算公式：

$$R = \frac{U_i - U_Z}{I_Z + I_L}$$

- 选 $R$ 需保证：
  - 电网最高、负载最轻时 $I_Z$ 不超过 $I_{Zmax}$（稳压管不烧毁）
  - 电网最低、负载最重时 $I_Z > I_{Zmin}$（稳压管不退出击穿区）

### 5. 开关型稳压电源（填空 94）

- 调整管工作在**开关**状态（饱和/截止交替），而非线性放大状态
- 效率**高于**线性稳压电源（典型 80%~95%，线性一般 40%~60%）
- 通过控制开关占空比（PWM）调节输出电压
- 缺点：纹波大、有开关噪声、EMI 较强

### 6. 故障诊断（填空 91）

| 故障现象 | 判断 | 原因 |
|----------|------|------|
| 输出正常 + **100Hz 纹波** | **滤波**环节失效 | 50Hz 市电经全波/桥式整流→100Hz 脉动，滤波电容开路 |
| 输出正常 + **50Hz 纹波** | 整流有一路开路 | 退化为半波整流 |
| 输出电压为 0 | 变压器或整流故障 | — |

- **填空91**：最可能是**滤波**环节失效，而**稳压**环节仍在正常工作

## 重点结论 / 公式速记

| 项目 | 公式/结论 |
|------|----------|
| 电源四大组成 | 变压器 → 整流 → 滤波 → 稳压 |
| 桥式整流（无滤波） | $U_o = 0.9\,U_2$ |
| 桥式整流 + 电容滤波（空载） | $U_o \approx \sqrt{2}\,U_2$ |
| 滤波电路三类 | 电容、电感、LC |
| 串联稳压四部分 | 取样、基准、比较放大、调整管 |
| 限流电阻公式 | $R = (U_i - U_Z)/(I_Z + I_L)$ |
| 开关电源调整管状态 | **开关**状态 |
| 开关电源效率 | **高于**线性电源 |
| 100Hz 纹波 | **滤波**环节失效 |
| 50Hz 纹波 | 整流二极管开路（半波） |
| 串联稳压本质 | **电压串联负反馈** |
| 变压器作用 | **降压** |
| 整流作用 | 交流→**直流** |

## 易错点提醒（对应题号）

| 题号 | 陷阱描述 | 正确结论 |
|------|----------|----------|
| **填空87–89** | 四大部分名称和顺序 | 变压器→**整流**→滤波→**稳压** |
| **填空88** | 各部分功能 | 变压器=电压转换；整流=电流方向统一；滤波=波形平滑；稳压=电压恒定 |
| **填空90** | 滤波三类 | 电容、**电感**、**LC（π型）** |
| **填空91** | 100Hz 纹波原因 | **滤波**失效，**稳压**正常（50Hz 全波整流→100Hz） |
| **填空92** | 串联稳压信号流向 | **取样**输出与**基准电压**在**比较放大器**输入端比较 |
| **填空93** | 限流电阻作用 + 与负载关系 | **限流**调压；与负载**串联** |
| **填空94** | 开关电源调整管状态 + 效率 | **开关**状态；效率**高于**线性 |
| **综合填空95** | 变压器/整流作用 | 变压器=**降压**；整流=**变直流** |
| **通用** | 串联稳压调整管工作区 | **放大区**（线性），不是开关状态 |
