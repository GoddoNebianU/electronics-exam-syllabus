# 第9章 直流稳压电源

> **题型分布**：填空 87–95（含综合填空95）
> **重要度**：★★（纯填空章，但概念密集、易考综合）

## 核心知识点

### 1. 直流稳压电源四大组成部分（填空87、88、89、综合95）
按顺序依次为：
1. **电源变压器**：把 220 V 交流市电降压到所需交流电压（电压转换的初始任务）。
2. **整流电路**：把交流转成脉动直流（统一电流方向），利用二极管单向导电性。
3. **滤波电路**：把脉动直流平滑为较平直的直流（使电流波形平滑）。
4. **稳压电路**：保证输出电压恒定（不随电网/负载波动）。

四部分缺一不可，是线性稳压电源的标准结构（填空87明确"线性稳压电源即串联型稳压电源"）。

### 2. 滤波电路（填空90、综合95）
- 常用三类：**电容滤波、电感滤波、LC 滤波**。
- **电容滤波**：与负载**并联**，空载时 `Uo ≈ √2 U2`；带载时介于 `0.9 U2 ~ √2 U2` 之间。
- **电感滤波**：与负载**串联**，利用电感阻碍电流变化，适用于大电流负载。
- **LC 滤波**：电感 + 电容组合，兼顾两者优点，纹波更小。
- 电容滤波简单常用，但带载能力差；电感滤波体积大但带载能力强。

### 3. 串联反馈式稳压电路（填空92）
四大组成部分：
1. **取样电路**：从输出端分压取样（电阻分压网络）。
2. **基准电压**：通常由稳压管提供（恒定参考）。
3. **比较放大**：把取样电压与基准电压比较，差值放大（常用运放或单管放大）。
4. **调整管**：根据比较放大结果控制导通程度，调节输出（工作在放大区，串联在主回路）。

信号流向：**取样电路**的输出与**基准电压**在**比较放大器**的输入端汇合比较，差值被放大后去控制**调整管**。
- 本质是一个**电压串联负反馈**闭环系统，可与第5章反馈知识对照理解。

### 4. 硅稳压管稳压电路（填空93）
- 限流电阻作用：**限制电流**使稳压管工作在反向击穿区，同时与负载**串联**连接分压。
- 公式：

$$R = \frac{U_i - U_Z}{I_Z + I_L}$$

- 选 `R` 要保证：
  - 电网最高、负载最轻时 `IZ` 不超过 `IZmax`（稳压管不烧毁）。
  - 电网最低、负载最重时仍能维持 `IZ > IZmin`（稳压管不退出击穿区）。
- 优点：电路简单；缺点：输出电流小、稳压精度低、输出电压固定。

### 5. 开关型稳压电源（填空94）
- 调整管工作在**开关**状态（饱和/截止交替），而非线性放大状态。
- 效率**高于**线性稳压电源（典型 80%~95%，线性稳压一般 40%~60%）。
- 通过控制开关占空比（PWM）调节输出电压。
- 缺点：纹波大、有开关噪声、电路复杂、EMI 较强。

### 6. 故障诊断（填空91）
- 输出电压正常但带明显 **100 Hz 纹波**（50 Hz 市电经全波整流后为 100 Hz 脉动）。
- 判断：**滤波**环节失效（电容开路或容量下降），而**整流**环节仍正常工作。
- 若纹波是 50 Hz，则可能是整流二极管有一路开路（退化为半波整流）。
- 若输出电压为 0，则可能是变压器或整流环节故障。

### 7. 四部分功能对照速记
| 环节 | 输入 | 输出 | 关键元件 |
|------|------|------|----------|
| 变压器 | 220 V AC | 低压 AC | 铁芯线圈 |
| 整流 | 低压 AC | 脉动 DC | 二极管（桥式） |
| 滤波 | 脉动 DC | 较平滑 DC | 电容/电感 |
| 稳压 | 较平滑 DC | 恒定 DC | 调整管/稳压管 |

## 重点结论 / 公式速记

| 项目 | 公式/结论 |
|------|----------|
| 电源四大组成 | 变压器 → 整流 → 滤波 → 稳压 |
| 桥式整流（无滤波） | `Uo = 0.9 U2` |
| 桥式整流 + 电容滤波（空载） | `Uo ≈ √2 U2` |
| 滤波电路三类 | 电容、电感、LC |
| 串联稳压四部分 | 取样、基准、比较放大、调整管 |
| 限流电阻公式 | `R = (Ui − UZ)/(IZ + IL)` |
| 开关电源调整管状态 | 开关状态 |
| 开关电源效率 | 高于线性电源 |
| 100 Hz 纹波 | 滤波环节失效 |
| 50 Hz 纹波 | 整流二极管开路（半波） |
| 串联稳压本质 | 电压串联负反馈 |

## 易错点提醒
- 100 Hz 纹波：我国市电 50 Hz，**全波/桥式**整流后脉动频率为 **100 Hz**（半波整流才是 50 Hz）；出现 100 Hz 纹波说明整流正常、滤波失效。
- 串联稳压电路"调整管"工作在**放大区**（线性），不要与开关电源混淆。
- 限流电阻与负载是**串联**（不是并联）；稳压管与负载是反向并联，但**限流电阻必须串**在主回路。
- 开关电源调整管是**开关**状态，效率**高于**线性稳压电源（填空94两空易混答）。
- 滤波三大类不要漏写"LC 滤波"。
- 串联反馈式稳压本质是**电压串联负反馈**闭环系统，可与第5章反馈知识对照理解。
- "变压器作用是降压，整流作用是变直流"（综合填空95）是固定答案，不可写反。
- 稳压管稳压电路只能输出**固定**电压（等于 UZ），不能调节；串联稳压可调。
