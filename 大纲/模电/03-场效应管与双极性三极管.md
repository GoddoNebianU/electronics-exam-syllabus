# 第3章 场效应管与双极性三极管

> **题型分布**：填空 19–40、MOS 管放大题 1–4、BJT 放大题 1–4、频率响应题 1–4
> **重要度**：★★★（题库篇幅最大，含 12 道含图计算题）

## 核心知识点

### 1. BJT 与 FET 的本质区别（填空19、20、25、26）
| 对比项 | BJT 双极型三极管 | FET 场效应管 |
|--------|------------------|--------------|
| 控制方式 | **电流**控制器件 | **电压**控制器件 |
| 载流子 | 两种（电子+空穴） | 一种（多子） |
| 输入电阻 | 小 | **大** |
| 控制量 | `iB` 控制 `iC` | `uGS` 控制 `iD` |

### 2. BJT 三种组态与三工作区（填空21、23、24）
- 三种组态：**共射、共集、共基**。
- 输出特性三区：**截止区、放大区、饱和区**。
- 放大区偏置条件：**发射结正偏、集电结反偏**（填空21）。

### 3. 静态工作点与非线性失真（填空28、29、30）
- Q 点**偏高**（接近饱和）→ **饱和失真**，输出底部削平。
- Q 点**偏低**（接近截止）→ **截止失真**，输出顶部削平。
- 共射电路输出底部削平属**饱和失真**，可**增大**基极偏置电阻 `Rb` 降 `IB` 消除。

### 4. 共集电极电路（射极跟随器，填空31、96）
- 又名**射极跟随器**。
- 电压放大倍数 `Av ≈ 1`（略小于 1，同相）。
- 输入电阻**大**，输出电阻**小**。常用于输入级、输出级、缓冲级。

### 5. 复合管（填空22）
两 β 相同晶体管组成复合管（达林顿）：电流放大系数 `β ≈ β1 · β2`。

### 6. Q 点稳定：分压式偏置（填空32、33）
- 影响 Q 点的主要因素：**温度**变化。
- 稳定措施：引入**直流负反馈**。
- 分压式偏置中发射极电阻 `Re` 引入**直流电流负反馈**，稳定 `ICQ`。

### 7. 多级耦合方式（填空34、35、36、37）
| 耦合方式 | 特点 |
|----------|------|
| **阻容耦合** | 各级 Q 点独立、结构简单，低频电压放大常用；不能放大直流 |
| **直接耦合** | 前后级 Q 点相互影响，存在**零点漂移**问题；集成运放内部必用 |
| **变压器耦合** | 可实现阻抗匹配，体积大、频率特性差 |

- 多级总电压放大倍数 = 各级电压放大倍数之**积**。
- 多级总通频带**窄于**单级。

### 8. 频率响应要点（填空38、39、40）
- 通频带 `BW = fH − fL`。
- 在 `fL`、`fH` 处增益比中频下降 **3 dB**，附加相移约 **±45°**。
- 高频段增益下降原因：**结电容、分布电容**。

## 重点结论 / 公式速记

| 项目 | 公式/结论 |
|------|----------|
| 复合管 β | `β ≈ β1·β2` |
| 共集 Av | `≈ 1`（同相） |
| 共射 Av | 相位**反**（输入输出相差 180°，填空96） |
| 放大区偏置 | 发射结正偏、集电结反偏 |
| 多级增益 | 各级增益之积 |
| 高频衰减来源 | 结电容 + 分布电容 |
| MOS 管 g m | `gm = 2·√(Kn·IDQ)`（饱和区） |

### MOS 管小信号参数（MOS 题1–4）
- 跨导：`gm = 2 Kn (VGSQ − VTN) = 2·√(Kn·IDQ)`
- 输出电阻：`rds = 1/(λ·IDQ)`（题中常给 `λ = 0`，即 `rds → ∞` 可视为开路）
- 共源电压增益：`Av = −gm·(Rd ∥ RL)`

### BJT 小信号参数（BJT 题1–4）
- `rbe = rbb' + (1+β)·(26 mV / |IEQ| mA)`
- 共射增益：`Av = −β·(Rc ∥ RL) / rbe`

## 易错点提醒
- 共射输出底部削平是**饱和**失真（不是截止）；判断关键看 Q 点偏高还是偏低。
- "增大 Rb 消除饱和失真"是因为 `IB = (VCC − VBE)/Rb`，`Rb` 增大 → `IB` 减小 → Q 点下移。
- 直接耦合才有**零点漂移**，阻容耦合各级 Q 点独立不会漂移传到下一级。
- MOS 管题若给 `λ = 0`，画交流等效时 `rds` 视为**开路**（无穷大），不要画电阻。
- BJT 题 4（开关 S 接 A/B/C）是判断工作区域：截止/放大/饱和三态，靠 `IB` 与 `ICSS` 比较判定。
- 频率响应题常给波特图，要会读转折点（即上下限频率）和每十倍频 −20 dB 的斜率。
