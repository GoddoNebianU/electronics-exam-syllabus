# 第4章 模拟集成电路

> **题型分布**：填空 41–47、差分放大电路题 1–4（深度负反馈下闭环增益）
> **重要度**：★★（题量适中，但与第5章反馈、第6章运放高度联动）

## 核心知识点

### 1. 集成运放的组成（填空41、42、46）
集成运放采用三级结构，全部使用**直接耦合**（集成电路无法做大电容）：
- **输入级**：通常采用**差分放大电路**，目的是减小**零点漂移**、提高**共模抑制比（CMRR）**。
- **中间级**：主要提供**电压放大**（高增益），常采用有源负载。
- **输出级**：主要提供**功率驱动**（低输出电阻、大电流），常用互补推挽电路。
- 偏置电路：采用**电流源**为各级提供稳定工作点。

### 2. 差分放大电路的核心概念（填空43、44、47）
- **差模信号**：两管输入端信号**大小相等、极性相反**（有用信号）。
- **共模信号**：两管输入端信号**大小相等、极性相同**（干扰、温度漂移等同相成分）。
- 差放对**差模**信号有较强放大能力，对**共模**信号有较强抑制能力（填空47）。
- 两管静态工作点不一致 → **共模**抑制能力下降，**差模**信号放大效果受影响（填空44）。
- 抑制共模原理：`Re`（或恒流源）对共模信号引入强负反馈，对差模信号则因两管电流变化相互抵消而无影响。

### 3. 共模抑制比 CMRR（综合填空97）
$$CMRR = \left|\frac{A_d}{A_c}\right|$$
- `Ad` 为差模增益，`Ac` 为共模增益。
- CMRR 越大，抗共模干扰（如温度漂移、50 Hz 工频、电机干扰）能力越强。
- 常用分贝表示：`CMRR(dB) = 20 lg|Ad/Ac|`。

### 4. 电流源的作用（填空45）
电流源电路常作为放大电路的**有源负载**或为放大电路提供**稳定的偏置电流**。
- 用有源负载替代大电阻，可在集成电路中获得高交流等效电阻而不占芯片面积。
- 用恒流源替代差放的发射极电阻 `Re` 是抑制零点漂移**最有效**的方法（差分题1）。
- 常见电流源电路：镜像电流源、比例电流源、微电流源、威尔逊电流源。

### 5. 零点漂移及其抑制（填空42、差分题1、综合97）
- **零点漂移**：输入为零时输出电压仍缓慢变化的现象，主要由温度变化和元件老化引起。
- 直接耦合放大电路各级漂移会被后级放大，是最严重的问题（集成运放必用直接耦合，因此必须解决）。
- 抑制方法：
  - 差分放大电路利用**对称性**抵消双端输出的漂移（综合97）。
  - 用恒流源替代 `Re` 提高共模抑制（最有效）。
  - 调整两管对称性，保证静态工作点一致。

### 6. 差分电路题解题要点（差分题2、3、4）
- 电压增益、共模抑制比等指标主要与**输出方式**（单端/双端）和**输入方式**（单端/双端）有关。
- **双端输出**：共模增益理论为 0，CMRR 趋于无穷；差模增益为单级的 2 倍。
- **单端输出**：共模增益受 `Re`（或恒流源内阻）限制；差模增益为双端的一半。
- 增大偏置电流源动态电阻 → 共模抑制能力**增强**、差模增益基本不变。
- 深度负反馈条件下（题3满足 `(1+AF) ≫ 1`）：闭环电压增益 `Avf ≈ 1/F`，直接由反馈网络电阻比决定。
- 题4：要会找级间交流反馈支路，判极性与组态，再用深度负反馈公式求闭环增益表达式。

### 7. 四种差放输入输出组合增益对比
| 输入\输出 | 双端输出 | 单端输出 |
|-----------|----------|----------|
| 双端输入（差模） | `Ad`（满增益） | `Ad/2` |
| 单端输入（差模） | `Ad`（满增益） | `Ad/2` |
| 共模输入（任意方式） | `Ac ≈ 0` | `Ac`（受 Re 限制） |

## 重点结论 / 公式速记

| 项目 | 公式/结论 |
|------|----------|
| 共模抑制比 | `CMRR = |Ad/Ac|` |
| 差模输入 | 两管输入大小相等、极性相反 |
| 共模输入 | 两管输入大小相等、极性相同 |
| 深度负反馈闭环增益 | `Avf ≈ 1/F` |
| 集成运放输入级 | 差分放大电路 |
| 集成运放耦合方式 | 直接耦合 |
| 抑制零漂最有效方法 | 恒流源替代 `Re` |
| 电流源两大用途 | 有源负载、稳定偏置 |
| 双端输出 CMRR | 理论为无穷大 |

## 易错点提醒
- 差模信号"大小相等极性相反"，共模是"大小相等极性相同"，**别记反**。
- "差放对共模信号放大能力弱"是因为 `Re`（或恒流源）对共模信号引入强负反馈，对差模信号则无影响（差模电流在 `Re` 上相互抵消）。
- 单端输出与双端输出的差模增益差 **2 倍**关系（双端输出是单端的 2 倍）。
- CMRR 是**比值**（无量纲），常用分贝表示 `20 lg CMRR`。
- 题中"满足深度负反馈"是关键提示，**不要**再算开环增益 `A`，直接用 `Avf ≈ 1/F`。
- 调整偏置电流源动态电阻影响的是**共模抑制**，不要误以为能改变差模增益。
- 集成运放"中间级提供电压放大、输出级提供功率驱动"，**不要**把两级功能记反。
- 差放抑制零漂靠的是**电路对称性**和**共模负反馈**，不是温度补偿。
- 集成运放必须采用**直接耦合**，因为集成电路无法制造大容量电容（填空42）。
