# 第8章 功率放大电路

> **题型分布**：填空 79–86、功率放大电路题 1–4
> **重要度**：★★★（概念+计算并重，含最大输出功率与效率计算）

## 核心知识点

### 1. 功率放大电路的任务与特点（填空79、综合99）
- 主要任务：向负载提供**足够大的信号功率**（不是电压放大）。
- 通常工作在**大信号**状态（与电压放大电路的最大区别，综合填空99）。
- 核心考核指标：**最大输出功率 Pom** 与**转换效率 η**（填空84）。
- 功放管常接散热片，防止**温度过高**损坏器件（填空86）。
- 与电压放大的区别：功放追求功率与效率，电压放大追求增益与线性度。

### 2. 功放分类（导通角，填空80、81、83）
| 类型 | 导通角 θ | 静态电流 | 静态功耗 | 效率（理想） |
|------|----------|----------|----------|--------------|
| **甲类（A 类）** | 360°（全周期） | 大 | 大 | 最高 50%（变压器耦合）/ 25%（无变压器） |
| **乙类（B 类）** | 180°（半周期） | 理论为零 | 小（理论为零） | 最高 **78.5%（π/4）** |
| **甲乙类（AB 类）** | 略大于 180° | 微小 | 介于二者 | 接近乙类 |

- 填空83：甲乙类静态功耗**小于**甲类、**大于**乙类。

### 3. 乙类互补推挽（OTL/OCL）功放（填空82、85、功放题1–4）
- 由**两只互补管**（NPN + PNP）轮流导通，正负半周各放大一半。
- **交越失真**：两管交替处（接近 0 V）由于死区电压（约 0.5~0.7 V）出现波形断裂（填空82）。
- 消除方法：使功放工作在**甲乙类**状态，给两管设微小静态偏置（微导通）。
- 偏置电路：常用**二极管 D1、D2** 或 **R3、R4 + 偏置三极管 T3** 提供压降（功放题2、4）。
- 理想最高效率 `η_max = π/4 ≈ 78.5%`（填空85）。
- 题1 优点：效率高、功放管静态功耗小；缺点：存在交越失真；克服：工作在甲乙类。

### 4. 最大输出功率计算（功放题2、4）
设电源电压 `VCC`，负载 `RL`，管饱和压降 `VCES`：
- OCL（双电源）最大输出电压幅值 `Uom = VCC − VCES`
- 最大输出功率：

$$P_{om} = \frac{(V_{CC} - V_{CES})^2}{2 R_L}$$

- 直流电源提供的平均功率：

$$P_{V} = \frac{2(V_{CC} - V_{CES}) \cdot V_{CC}}{\pi R_L}$$

- 转换效率：

$$\eta = \frac{P_{om}}{P_V} = \frac{\pi}{4} \cdot \frac{V_{CC} - V_{CES}}{V_{CC}}$$

**题2 数据**：`VCES = 3 V, VCC = 15 V, RL = 8 Ω` → `Uom = 12 V` → `Pom = 12²/(2·8) = 9 W`。
**题4 数据**：`UCES = 2 V`，代入公式类似计算。

### 5. 静态工作点（功放题2、4）
- 互补对称乙类/甲乙类 OCL 电路**静态时发射极电位 UEQ = 0**（双电源对称）。
- OTL 单电源电路静态 `UEQ = VCC/2`（电容耦合）。
- **R1 虚焊（开路）**（题2）：上半部偏置断 → 上管截止 → 输出只剩负半周，正半周削平。
- **R3、R4、T3 的作用**（题4）：提供约 1.4 V 偏置电压，使 T1、T2 微导通，消除交越失真。

## 重点结论 / 公式速记

| 项目 | 公式/结论 |
|------|----------|
| 乙类最高效率 | `η = π/4 ≈ 78.5%` |
| 甲类导通角 | 360° |
| 乙类导通角 | 180° |
| 甲乙类导通角 | 略大于 180° |
| OCL 最大输出功率 | `Pom = (VCC−UCES)² / (2 RL)` |
| 消除交越失真 | 工作于甲乙类（微导通偏置） |
| 静态 UEQ（OCL） | 0 V |
| 效率最高类别 | 乙类（理论 π/4） |
| 二极管 D1/D2 作用 | 提供偏置消交越失真 |
| 甲类效率上限 | 50%（变压器耦合） |

## 易错点提醒
- "效率最高"是**乙类**（78.5%），不是甲类（25%/50%）；填空易错。
- "消除交越失真"选**甲乙类**，不是乙类；乙类本身就有交越失真。
- 计算 `Pom` 时记得减去**饱和压降 VCES**，不要直接用 `VCC²/2RL`（那是忽略 VCES 的理想值）。
- 静态时 UEQ = 0 是 OCL 双电源电路的特征；OTL 单电源电路 UEQ = VCC/2。
- 二极管 D1、D2 的作用是**提供约 1.4 V 偏置**使两管微导通，不是整流也不是稳压。
- 题3 "哪类效率最高？哪类消除交越失真？"——两个问题答案**不同**：乙类最高，甲乙类消交越失真，**别混答**。
- 功放管散热片是为**降温保护器件**，不是为提高效率。
- 甲乙类静态功耗**大于**乙类（因为有微小静态电流），不要答反（填空83）。
- 互补对称电路必须用 NPN + PNP 互补管，不能两只同型管（否则无法正负半周轮流）。
