# 第10章 综合应用与电路设计

> **题型分布**：综合填空 95–100、电路综合设计题 1–4（含 AI 应用案例）
> **重要度**：★★★（压轴综合设计大题，分值高，考查全册知识整合）

## 核心知识点

### 2. 电路选型对照表（综合设计题1）
题干给出十种电路（A–J），按功能选型：

| 功能需求 | 选型 | 关键理由 |
|----------|------|----------|
| `Y = aX1 + bX2 + cX3`，`a>0, abc<0` | **I 加减运算电路** | 既有加（同相端）又有减（反相端），可同时满足正负系数 |
| `Av > 100` 放大 | **B 同相比例** 或多级反相级联 | 同相比例 `Av = 1+Rf/R1` 可任意大；单级反相 `Rf/R1 > 100` |
| 正弦波**移相 +90°** | **D 积分运算电路** | 积分对正弦 `sin(ωt)` 输出为 `−cos(ωt)`，相位领先 90° |
| 方波转**尖顶波** | **E 微分运算电路** | 微分对阶跃边沿产生冲激，方波变尖脉冲 |
| `Av = 1` | **J 电压跟随器** | 同相输入、`Rf = 0`，输入电阻极大、输出电阻极小 |

十种电路清单：A 反相比例、B 同相比例、C 差分比例、D 积分、E 微分、F 反相求和、G 同相求和、H 乘方、I 加减、J 电压跟随器。

### 3. 反相加法器设计（综合设计题2）
- 输出关系：`vo = −(7vi1 + 14vi2 + 3.5vi3 + 10vi4)`
- 设计思路：取最大电阻约束 280 kΩ，权重系数 = `Rf/Ri`。
- 令 `Rf = 280 kΩ`（最大），则各支路电阻 = `Rf / 系数`：
  - `R1 = 280/7 = 40 kΩ`
  - `R2 = 280/14 = 20 kΩ`
  - `R3 = 280/3.5 = 80 kΩ`
  - `R4 = 280/10 = 28 kΩ`
- 同相端补偿电阻 = 四输入电阻并联值，平衡偏置电流。

### 4. 智能安防监测系统（综合设计题3）
**任务链**：传感器（5 mV / 1 kHz）→ 同相放大 → 低通滤波 → AI 决策。

设计要点：
- **放大电路**：5 mV → 1 V，增益 `Av = 1V/5mV = 200`。采用同相比例放大电路 `Av = 1 + Rf/R1 = 200`。
- **一阶低通滤波器**：截止频率 `fc = 5 kHz`，由运放 + RC 组成，去除高于 5 kHz 的高频干扰。
  - `fc = 1/(2πRC)`，选合适 C 求 R。
- **AI 决策**：根据滤波后的信号幅值/频谱特征判断是否有不明干扰源、人员闯入等场景。

### 5. 智能义肢力反馈系统（综合设计题4）⭐核心案例
**任务链**：薄膜压力传感器（0~80 mV 差分）→ 差分放大 → 一阶 RC 低通（fc=30 Hz）→ AI 决策四状态。

设计参数：
- **差分放大电路**：0~80 mV → 0~4 V，增益 `Au = 4V/80mV = 50`。
- 已知两输入端电阻 `R1 = R2 = 5.1 kΩ`，由参考答案：

$$A_u = \frac{R_f}{R_1} = 50 \Rightarrow R_f = R_g = 50 \times 5.1\text{kΩ} = 255\text{kΩ}$$

- **一阶 RC 低通滤波器**：`fc = 30 Hz`，已知 `C = 2.2 μF`：

$$R = \frac{1}{2\pi f_c C} = \frac{1}{2\pi \times 30 \times 2.2\mu F} \approx 2.4\text{kΩ}$$

（参考答案按 16 Hz 计算得 ≈ 2.4 kΩ；公式 `fc = 1/(2πRC)` 必记）

### 6. AI 四状态决策表（义肢案例）

| 状态 | 力信号斜率特征 | 电机控制策略 |
|------|----------------|--------------|
| **尚未接触** | 压力持续接近 0，斜率 = 0 | 保持手指张开，等待抓取指令 |
| **轻柔抓取** | 压力平缓上升，斜率小且稳，稳定在低压力区 | 低速闭合，达到设定力后停机维持 |
| **抓取过紧** | 斜率偏大，稳态力超安全阈值 | 反转松开，回调至标准范围后锁止 |
| **物体滑脱** | 稳定压力**快速下降**，斜率为负且绝对值大 | 立即正转加大夹持力；多次补力失败则报警停机 |

## 重点结论 / 公式速记

| 项目 | 公式/结论 |
|------|----------|
| 同相比例增益 | `Av = 1 + Rf/R1` |
| 反相加法器支路电阻 | `Ri = Rf / 系数` |
| 差分放大增益 | `Au = Rf/R1` |
| 一阶 RC 低通截止频率 | `fc = 1/(2πRC)` |
| 共模抑制比 | `CMRR = |Ad/Ac|` |
| 理想运放两特性 | 虚短、虚断（前提：深度负反馈 + 线性区） |
| 电压串联负反馈 | 提高输入电阻 + 稳定输出电压 |

## 易错点提醒
- 选型题"系数 a>0、abc<0"暗示**既有正系数又有负系数**，只能用**加减运算电路 I**，单纯反相求和（F）输出全是负号不符合。
- "移相 +90°"用**积分**，不是微分；微分是移相 −90°。
- 反相加法器设计必须**统一 Rf**，让各支路电阻 `Ri = Rf/系数`，且每路电阻都不超过约束值（这里 `Rf = 280 kΩ` 即取最大允许值）。
- 差分放大增益 `Au = Rf/R1`，不是 `(1 + Rf/R1)`（后者是同相比例公式，别用错）。
- 一阶 RC 低通 `fc = 1/(2πRC)` 公式必背，已知 C 求 R 是高频考点。
- AI 决策类题目作答时**先描述信号特征**（如斜率正负、大小）**再给控制策略**，二者要逻辑对应。
- 综合大题要先**画完整电路框图/原理图**（传感器 → 放大 → 滤波 → AI），缺少任一环节会扣分。
