# 第9章 数模与模数转换

> **题型分布**：判断 55–60、填空 44–50
> **重要度**：★★☆（概念+公式计算题，无大题，但 ADC/DAC 方向辨析多）

## 核心知识点

### 1. DAC 与 ADC 功能辨析（判断55–60 重点）
- **DAC（数模转换器）**：将**数字量**转换为**模拟量**（填空44）
- **ADC（模数转换器）**：将**模拟量**转换为**数字量**（判断55、60 ✓）
- **判断56、58、59陷阱**：把 DAC 说成"模拟→数字"是**错**的（反向命题，高频陷阱）
- **判断57**：A/D 取样是将**时间上连续**的模拟量转换为**时间上断续**的模拟量（幅值仍连续）✓

### 2. DAC 主要参数（填空44、45、46）
- **分辨率**：最小输出电压（对应输入数字量 1）与最大输出电压之比
  - 公式：`分辨率 = 1 / (2ⁿ − 1)`，n 为位数
- 用最小量化单位表示为满量程的分数：
  - 8 位 DAC：`1 / (2⁸ − 1) = 1/255`（填空45）
  - 10 位 DAC：`1 / (2¹⁰ − 1) = 1/1023`（填空46）
- 分辨率也可用位数 n 直接表示（n 越大越精细）

### 3. DAC 电路结构（填空47）
- **权电阻网络 DAC**：
  - **缺点**：电阻种类多、阻值差异大（如 2⁰R、2¹R…2ⁿ⁻¹R），**难以集成**且精度难保证
- **倒 T 型电阻网络 DAC**（R-2R 梯形）：
  - **优点**：只用 R 和 2R 两种电阻，**易于集成**；无论开关位置，各支路电流恒定，**转换速度快**
  - 是目前集成 DAC 中应用最广的电路形式

### 4. A/D 转换四步（填空44）
A/D 转换过程分四步：
```
取样 → 保持 → 量化 → 编码
```
- **取样**：时间上连续→断续（需满足取样定理 `f_s ≥ 2 f_max`）
- **保持**：保持取样值一段时间，供后续量化
- **量化**：将取样电压归类到有限个离散电平
- **编码**：将量化结果用二进制代码表示

### 5. ADC 类型对比（填空48、49、50）
| 类型 | 转换速度 | 抗干扰 | 特点 |
|------|----------|--------|------|
| **并联比较型**（Flash） | **最快** | 一般 | n 位需 2ⁿ−1 个比较器，电路庞大 |
| **逐次逼近型**（SAR） | 中等 | 一般 | n 位需 n 个时钟周期完成一次转换 |
| **双积分型** | **最慢** | **最强** | 对周期性干扰有极强抑制 |

- 速度排序：**并联比较型 > 逐次逼近型 > 双积分型**（填空48）
- **n 位逐次逼近型 ADC**：完成一次转换至少需要 **n 个时钟周期**（填空49，10 位需 10 个）
- **双积分型 ADC 工作两阶段**（填空50）：
  - 第一阶段：对**输入电压**进行**定时**积分（固定时间 T₁）
  - 第二阶段：对**基准电压**进行**定值**反向积分（直至输出过零，时间 T₂ 正比于输入）

## 重点结论 / 公式速记

| 项 | 结论 / 公式 |
|----|-------------|
| DAC 功能 | 数字量 → 模拟量 |
| ADC 功能 | 模拟量 → 数字量 |
| DAC 分辨率 | `1 / (2ⁿ − 1)` |
| 8 位 DAC 分辨率 | `1/255` |
| 10 位 DAC 分辨率 | `1/1023` |
| A/D 四步 | 取样、保持、量化、编码 |
| 逐次逼近 n 位周期数 | `n` |
| ADC 速度排序 | 并联比较 > 逐次逼近 > 双积分 |

- 权电阻网络缺点：电阻种类多、难集成
- 倒 T 型网络优点：只用 R 和 2R，易集成、速度快
- 双积分 ADC 第一阶段：对输入电压定时积分；第二阶段：对基准电压定值积分

## 易错点提醒
- **判断56、58、59**：DAC 是**数字→模拟**，凡说成"模拟→数字"必错（高频反向命题）
- **判断60**：ADC 是**模拟→数字** ✓（注意与 DAC 区分）
- **填空44、45、46**：分辨率分母是 `2ⁿ − 1`（不是 2ⁿ），8 位是 255、10 位是 1023
- **填空47**：权电阻网络缺点是**电阻种类多/阻值差异大**；倒 T 型优点是**只用两种电阻、易集成**
- **填空48**：速度排序记牢"并联最快、逐次居中、双积分最慢"；抗干扰双积分**最强**
- **填空49**：n 位逐次逼近需 **n** 个时钟周期（10 位 = 10 个，不是 2ⁿ 或 n+1）
- **填空50**：双积分第一阶段对**输入电压**积分（不是基准），第二阶段才对**基准电压**反向积分，两阶段对象不能调换
- A/D 四步顺序：**取样→保持→量化→编码**，顺序不可颠倒
