# 第3章 门电路

> **题型分布**：判断 14–21、填空 15–20
> **重要度**：★★☆（概念题为主，无大题，但概念辨析多）

## 核心知识点

### 1. 正逻辑与负逻辑
- **正逻辑**：高电平 = 1，低电平 = 0（标准体制）
- **负逻辑**：高电平 = 0，低电平 = 1
- **判断14陷阱**：题中"高电平用1、低电平用0"应称为**正逻辑**，原题写"负逻辑"是错的
- 同一电路正逻辑下为"与门"，负逻辑下则变为"或门"
- **判断18陷阱**：输入全高时输出低、否则输出高 → 这是**与非门（NAND）**，不是与门

### 2. TTL 与 CMOS 对比（填空15重点）
| 项目 | TTL | CMOS |
|------|-----|------|
| 工作电源 | 典型 **5V** | 范围宽（3～18V） |
| 功耗 | 静态较大 | 静态**极低** |
| 抗干扰 | 一般 | 强 |
| 集成度 | 中 | 高 |
| 输入端 | 悬空≈高电平 | **不允许悬空** |
| 防静电 | 一般 | **特别注意防静电** |

- **TTL 与 CMOS 不可互换替代**（判断19 ✗）：电气参数（电平、功耗、驱动能力）不同
- **TTL 输入悬空**等效为**高电平**（判断20 ✗，原题说"低电平"是错的）
- **CMOS 输入悬空**电平**不确定**，易受干扰甚至损坏器件（判断21 ✗）

### 3. 特殊输出门电路
- **OC 门（TTL）/ OD 门（CMOS）**：
  - 输出端可并联使用，实现**线与**逻辑（填空18）
  - 必须外接**上拉电阻**接到电源
- **三态输出门（TS 门）**：
  - 除高电平、低电平外，第三种状态是**高阻态**（填空19）
  - 主要用于实现**总线传输**（判断16 ✓），允许多个输出分时共用一条总线
  - **判断15陷阱**：能实现"线与"的是 **OC/OD 门**，不是三态门；三态门用于总线
- **TTL 与非门输出端不能直接并联**（判断17 ✓），否则会烧毁器件

### 4. CMOS 电路结构与传输门
- CMOS 与非门：输入接低电平的支路，对应的 **NMOS 导通、PMOS 截止**（填空17）
- **CMOS 传输门（TG）**：
  - NMOS 栅极低电平、PMOS 栅极高电平时，传输门处于**截止（断开）状态**（填空20）
  - 控制信号反相时导通，可双向传输模拟或数字信号
- **TTL 与非门输入低电平电流 IIL 方向**：**流出**输入端（填空16）

## 重点结论 / 公式速记

| 器件 | 关键特性 |
|------|----------|
| TTL | 5V 电源、悬空=高、IIL 流出输入端、输出不可直接并联 |
| CMOS | 静态功耗极低、防静电、输入不允许悬空 |
| OC / OD 门 | 线与、需外接上拉电阻 |
| 三态门（TS） | 高阻态、总线传输、需使能控制 |
| 传输门（TG） | 双向开关、NMOS/PMOS 栅极互补控制 |

## 易错点提醒
- **判断14**：高=1低=0 是**正**逻辑，题干"负逻辑"是反命题
- **判断15、17**：线与用 OC/OD 门，三态门用于总线，二者功能不要混
- **判断18**："全高则低、否则高"是**与非门**，不是与门
- **判断19**：TTL 与 CMOS 即使逻辑功能相同也**不可互换**
- **判断20**：TTL 悬空是**高电平**（不是低）
- **判断21**：CMOS 悬空电平**不确定**（不是确定的高或低）
- **填空16**：IIL 方向是**流出**输入端（电流从输入端流入前级输出），不要写反
- **填空20**：传输门 NMOS 栅低 + PMOS 栅高 → **截止**（控制电平记反会答"导通"）
