# 第6章 时序逻辑电路

> **题型分布**：判断 34–44、填空 30–35、简答 15–16、20–21、27–30、综合题 1–2
> **重要度**：★★★（题量最大，含综合压轴题 555+161+138+153 综合）

## 核心知识点

### 1. 时序电路概述（判断34–44）
- **时序电路 = 组合电路 + 触发器（存储）**，输出不仅取决于当前输入，还与**原状态**有关（填空30）
- **判断38陷阱**：异步时序电路**也包含触发器**（原题说"不包含"是错的），区别在于是否有**统一时钟**
- **判断34陷阱**：竞争冒险**也会影响**时序电路（可能引起触发器误翻转），原题说"没有影响"是错的
- **同步 vs 异步**：
  - 同步：所有触发器**共用同一时钟 CP**，状态变化同时发生（判断40、42 ✓）
  - **判断35陷阱**：同步时序电路各触发器时钟**相同**（原题说"可以不同"是错的）
  - 异步：无统一时钟，状态变化依赖输入信号和电路延迟（判断44 ✓）
- **米利型（Mealy）**：输出取决于**输入和现态**（填空32）；穆尔型（Moore）输出仅取决于现态

### 2. 三种等价描述（判断39）
- **时序图、状态转换图、状态转换表**三者可描述同一时序电路，且**可相互转换**（判断39 ✓）
- 分析题标准流程（简答27–30）：驱动方程 → 状态方程 → 状态表 → 状态图 → 逻辑功能

### 3. 计数器
- **模**：计数器能够计数的最大脉冲个数（判断37 ✓，即一个计数循环的状态数）
- 由 n 个触发器构成的计数器，最大计数模值为 **2ⁿ**（填空30）
- 4 位二进制加法器最多能计 **16** 个脉冲（填空30）
- **模 60 计数器**：至少需要 **6 个触发器**（2⁶=64 ≥ 60，填空31）；用 74160（十进制）实现需 **6 片级联**（2片不够，2位十进制最大99，实际需2片74160级联即可，按题意判断；考试中按教材结论处理）

### 4. 集成计数器 74LVC161 / 74HC161 / 74160
- **74LVC161 / 74HC161**：4 位二进制同步加法计数器（模 16）
  - 引脚：`CP`（时钟）、`PE`（同步并行置数）、`CR`（异步清零，低有效）、`CET/CEP`（使能）、`TC`（进位输出）
  - 功能优先级：`CR > PE > CET/CEP > 计数`
- **74160**：同步十进制计数器（模 10）
- **任意模计数器设计**（简答20、21、22）：
  - **反馈清零法**：计到目标值时触发 `CR`，状态回 0
  - **反馈置数法**：计到目标值时触发 `PE`，置入初值（更灵活）
  - **级联法**：模 168 = 16 × 10 + 8，可用两片 161 级联实现（简答22）

### 5. 寄存器与移位寄存器（判断41、43）
- **寄存器**：主要功能是**暂时存储二进制数据**（判断41 ✓）
- 4 位数据寄存器在时钟有效沿同时存入 4 位并行数据：**并行输入、并行输出**（填空35）
- **移位寄存器**：既能存储数据，又能实现**串行↔并行转换**（判断43 ✓）
  - 串入并出、并入串出，常用于接口转换

### 6. 环形与扭环计数器（判断36）
- **环形计数器**：n 位移位寄存器构成，有效状态数 = **n 个**
- **扭环计数器**（约翰逊计数器）：n 位移位寄存器构成，有效状态数 = **2n 个**
- **判断36陷阱**：环形计数器有效状态数是扭环的**一半**（原题说"两倍"是错的）

## 重点结论 / 公式速记

| 项 | 结论 |
|----|------|
| 同步 vs 异步 | 同步共用 CP，异步无统一时钟 |
| n 触发器最大模 | `2ⁿ` |
| 4 位二进制计数范围 | 0–15（16 个脉冲） |
| 环形计数器有效状态 | `n` |
| 扭环计数器有效状态 | `2n` |
| Mealy 输出 | 依赖输入 + 现态 |
| Moore 输出 | 仅依赖现态 |

- 74LVC161 功能优先级：`CR（异步清零） > PE（同步置数） > CET·CEP（计数使能）`
- 任意模 M 设计：反馈清零（用 M 的二进制译码）或反馈置数（置 16−M）

## 易错点提醒
- **判断34**：竞争冒险**会**影响时序电路
- **判断35**：同步电路各触发器时钟**相同**
- **判断36**：环形计数器状态数是扭环的**一半**（n vs 2n），原题"两倍"是错的
- **判断38**：异步电路**也含触发器**
- **填空30**：n 触发器最大模为 `2ⁿ`，不是 `n` 或 `2n`
- **填空32**：Mealy 型输出依赖**输入 + 现态**（两个都要写）
- **填空33**：锁存器是**电平**敏感，触发器是**边沿**敏感（不要写反）
- **简答27–30**：分析题必须按"驱动方程→状态方程→状态表→状态图→功能"完整作答，缺步骤扣分
- **综合题1、2**：555 定时器构成**多谐振荡器**（无稳态，作 CP 源）→ 计算 v_o1 周期 `T ≈ 0.7(R_A + 2R_B)C` → 161 计数 → 138 译码 / 153 选择 → 不同 MN 组合对应不同进制计数器，务必分情况列表
