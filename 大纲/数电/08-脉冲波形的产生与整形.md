# 第8章 脉冲波形的产生与整形

> **题型分布**：判断 51–54、填空 41–43、简答 1–2、19
> **重要度**：★★★（含简答大题 + 综合题1、2 中 555 是核心，必考）

## 核心知识点

### 1. 555 定时器概述
- 一种模拟+数字混合的**中规模集成电路**，外接少量阻容元件可构成三种典型电路：
  - **多谐振荡器**（无稳态，方波源）
  - **单稳态触发器**（一稳态一暂稳态，定时/延时/整形）
  - **施密特触发器**（双稳态，整形/鉴幅）
- 内部由分压电阻链（3 个 5kΩ，得名 555）、两个电压比较器、一个 RS 触发器、一个放电管组成

### 2. 三种电路的稳态特征（判断51–54 重点）
- **多谐振荡器**：**没有稳定状态**，只有两个暂稳态，自动在两态间切换，输出方波（判断51 ✓）
- **单稳态触发器**：**一个稳定状态 + 一个暂稳态**（判断52、54 ✓）
  - 外触发进入暂稳态，经 `t_w` 后自动返回稳态
  - 暂稳态维持时间取决于 **RC 时间常数**（填空41）
- **施密特触发器**：**两个稳定状态**，有两个不同的触发电平，具有**回差（滞后）特性**（判断53 ✓，填空43）

### 3. 单稳态触发器（555 构成）
- 输出脉冲宽度：`t_w ≈ 1.1 RC`（填空41 必背）
- 触发：负脉冲加到 `TRIG`（2 脚）
- 放电管在暂稳态期间截止，电容 C 经 R 充电至 `2V_cc/3` 时翻转回稳态

### 4. 多谐振荡器（555 构成，填空42）
- **电容 C 充电回路**：经电阻 **R_A 和 R_B**（电源 → R_A → R_B → C → 地）
- **电容 C 放电回路**：只经电阻 **R_B**（C → R_B → 7 脚放电管 → 地）
- 振荡周期：`T = t_PH + t_PL ≈ 0.7(R_A + 2R_B)C`
  - 高电平时间 `t_PH ≈ 0.7(R_A + R_B)C`
  - 低电平时间 `t_PL ≈ 0.7 R_B·C`
- 占空比可调（取决于 R_A、R_B 比例）
- **综合题1、2**：555 构成多谐振荡器作为时钟源驱动 74LVC161，务必会用上述周期公式计算 `v_o1` 的周期

### 5. 施密特触发器（555 构成，简答19）
- 将 555 的 `TH`（6 脚）和 `TRIG`（2 脚）相连作为输入
- 上限阈值 `V_T+ = 2V_cc/3`，下限阈值 `V_T- = V_cc/3`
- 回差电压 `ΔV_T = V_T+ − V_T- = V_cc/3`
- 输入正弦波/三角波 → 输出矩形波，用于波形变换、整形、鉴幅

## 重点结论 / 公式速记

| 电路 | 稳态数 | 主要用途 |
|------|--------|----------|
| 多谐振荡器 | **0**（两个暂稳态） | 方波信号源 |
| 单稳态触发器 | **1** + 1 暂稳 | 定时、延时、整形 |
| 施密特触发器 | **2** | 波形变换、整形、鉴幅 |

| 555 应用电路 | 关键公式 |
|-------------|----------|
| 多谐振荡器周期 | `T ≈ 0.7(R_A + 2R_B)C` |
| 单稳态脉宽 | `t_w ≈ 1.1 RC` |
| 施密特阈值 | `V_T+ = 2V_cc/3`，`V_T- = V_cc/3`，`ΔV_T = V_cc/3` |

- 充电经 `R_A + R_B`，放电只经 `R_B`
- 三种电路均可由 **555 定时器**外接阻容元件构成（填空43）

## 易错点提醒
- **判断51、52、53、54** 均为稳态数辨析：多谐=**无**稳态、单稳=**一**稳态、施密特=**两**稳态，务必记牢
- **判断52、54** 是同一知识点重复命题（一个稳态 + 一个暂稳态），答案都是 ✓
- **填空41**：单稳态脉宽公式 **`t_w ≈ 1.1 RC`**（系数是 1.1，不是 0.7 或 1.0）
- **填空42**：充电回路经 **R_A、R_B** 两个电阻；放电回路**只经 R_B**（少一个电阻，易漏）
- **填空43**：三种电路均可由 **555 定时器**构成（不是 556 或其他）
- **综合题1、2** 中 555 一定是构成**多谐振荡器**作为时钟源，先用周期公式算 T，再分析 161 计数
- **简答1、2**：描述工作原理时区分"暂稳态"（自动返回）和"稳态"（需触发才离开），用词不可混
