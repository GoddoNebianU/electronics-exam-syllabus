# 第4章 组合逻辑电路

> **题型分布**：判断 22–28、填空 21–26、简答 8/12/13/14/17/18/23、综合设计 24–26
> **重要度**：★★★（综合大题核心：多数表决、液位检测、举重裁判；MSI 芯片分析题多）

## 核心知识点

### 1. 组合逻辑电路特点与设计步骤
- **特点**：任意时刻输出仅取决于该时刻输入，与电路原状态无关；**不存在反馈回路**（判断22 前半句 ✓）
- **设计四步**：列真值表 → 写逻辑表达式 → 化简（公式法/卡诺图） → 画逻辑电路图
- **分析步骤**：逐级写输出表达式 → 列真值表 → 说明逻辑功能

### 2. 竞争冒险（判断22–23、27 重点）
- **原因**：信号通过不同路径到达终点的时间有先后（门传输延迟不同），产生**竞争**，可能导致输出短暂出错
- **判断22陷阱**：组合电路**虽无反馈，但仍会出现竞争冒险**（原题说"不会出现"是错的）
- **判断23**：电路越简单、门越少，竞争冒险可能性越小 ✓
- **判断27陷阱**：有竞争**不一定**产生错误输出（仅有"临界竞争"才出错）
- **三种消除方法**（填空22）：
  1. 引入选通脉冲（在信号稳定后取样）
  2. 输出端加**滤波电容**
  3. 修改逻辑设计，添加**冗余项**（添加项）

### 3. 译码器（74LS138 / 74HC138）
- **3 线-8 线译码器**：3 位地址输入，8 个输出（低电平有效），每个输出对应一个最小项的非 `Ȳᵢ = m̄ᵢ`
- **判断24陷阱**：译码器和数据选择器**都属于组合逻辑电路**（原题说数据选择器是时序电路，错）
- **判断26陷阱**：译码器翻译的是**二进制代码**，不是"语言"
- **实现逻辑函数**（填空23、24，简答23）：
  - 将函数写成**最小项之和**形式 `F = ∑m(...)`
  - 译码器输出端 `Y₁、Y₃、Y₅、Y₇`（即对应的 m̄）接到**与非门**输入端
  - 例：`F(A,B,C) = ∑m(1,3,5,7)` → 将 `Y1, Y3, Y5, Y7` 接入与非门

### 4. 数据选择器（74151 / 74HC153）
- **八选一（74151）**：**3 个地址输入端**、**8 个数据输入端**（填空21）
- **四选一（74HC153）**：2 个地址端、4 个数据端
- **实现逻辑函数原理**（填空26）：地址输入端接**输入变量**，数据输入端接**常数（0/1）或剩余变量**
- **简答12、18**：给出 74151 接线图，分析输出 L（或 Z）的表达式

### 5. 加法器（判断25、28，填空25）
- **半加器**：仅考虑本位相加，不考虑低位进位
- **全加器**：既考虑低位进位，又考虑向高位的进位（判断28 ✓）
- **超前进位加法器**（判断25陷阱）：相对串行加法器的优点是**运算速度快**（提前算进位），不是"电路更简单"（实际更复杂）
- 四位超前进位加法器：
  - 进位产生变量 `Gi = Ai · Bi`
  - 进位传播变量 `Pi = Ai + Bi`

## 重点结论 / 公式速记

| MSI 芯片 | 功能 | 实现逻辑函数方法 |
|----------|------|------------------|
| 74LS138 / 74HC138 | 3-8 译码器 | 最小项非 → 与非门汇总 |
| 74151 | 八选一 MUX | 地址接变量，数据接常量 |
| 74HC153 | 双四选一 MUX | 同上 |

| 冒险消除法 | 关键词 |
|-----------|--------|
| 选通脉冲 | 信号稳定后取样 |
| 滤波电容 | 输出端并联电容 |
| 冗余项 | 添加项 `AB + ĀC + BC` |

- 超前进位：`Gi = Ai·Bi`，`Pi = Ai + Bi`
- 全加器本位和 `Si = Ai ⊕ Bi ⊕ Ci-1`，向高位进位 `Ci = Ai·Bi + (Ai⊕Bi)·Ci-1`

## 易错点提醒
- **判断22**：组合电路**会**有竞争冒险（虽无反馈）
- **判断24**：数据选择器是**组合**电路，不是时序电路
- **判断25**：超前进位优点是**速度快**，代价是电路更复杂
- **判断26**：译码器翻译二进制代码，不是语言
- **判断27**：竞争不一定会导致错误输出
- 综合设计题（24 多数表决、25 液位检测、26 举重裁判）务必按"真值表→表达式→化简→电路图"完整流程作答，**真值表不可省**
- 举重裁判（综合26）："主裁判 + 至少一名副裁判"逻辑 = `F = 甲·(乙 + 丙)`
- 用 138 实现函数时，输出是**最小项的非（低有效）**，外接**与非门**（不是或门）
