# 第2章 逻辑代数基础

> **题型分布**：判断 9–13、填空 8–14、简答 5–7（卡诺图化简含无关项）
> **重要度**：★★★（含 3 道简答大题，化简计算题必考）

## 核心知识点

### 1. 最小项与最大项
- **最小项**：n 变量逻辑函数共有 **2ⁿ 个**最小项（判断11 ✗，原题说 n 个是错的）；每个最小项含 **n 个因子**（判断10 ✓），且每个变量以原变量或反变量形式出现一次
- **最大项**：n 变量共有 **2ⁿ 个**最大项，每个最大项是 **n 个变量的或运算**（填空14）
- 最小项编号 `mᵢ` 与最大项编号 `Mᵢ` 互补：`mᵢ = M̄ᵢ`

### 2. 卡诺图化简
- 合并的最小项个数必须是 **2 的整数次幂**（1、2、4、8、16…）（判断9 ✓）
- **卡诺圈可以重叠**：每个方格可被多个圈包围（判断13 ✗，原题说"只能被一个圈包围"是错的）
- 圈越大越好、圈数越少越好；每个圈至少含一个其他圈未包含的最小项
- **无关项（约束项）**：在卡诺图中用 **`×`** 表示，化简时可任意视为 0 或 1（填空13）
- 表示法：`∑m(...)` 最小项之和，`∑d(...)` 无关项

### 3. 对偶式与反演规则（摩根定理）
- **对偶式 Y'**：把 `·` 与 `+` 互换、`0` 与 `1` 互换，**变量和长短不变**、运算顺序保持
  - 例：`Y = A + BC` → `Y' = A(B + C)`
  - 例：`Y = AB + C` → `Y' = (A + B)C`
- **反演规则（求反函数）**：在对偶基础上再把每个变量取反
  - 例：`Y = A + BC` → `Ȳ = Ā( B̄ + C̄)`
  - 例：`Y = AB + C` → `Ȳ = (Ā + B̄)C̄`
- **摩根定理**：`A·B = Ā + B̄`，`A + B = Ā · B̄`

### 4. 常用公式与化简
- 吸收律：`A + AB = A`，`A + ĀB = A + B`，`AB + ĀB = B`
- 冗余项（添加项）：`AB + ĀC + BC = AB + ĀC`
- 典型化简：`Y = AB + AC + BC` 不能直接化简（已是较简形式，按题目实际再处理）

## 重点结论 / 公式速记

| 概念 | 结论 |
|------|------|
| n 变量最小项数 | `2ⁿ` |
| 每个最小项因子数 | `n` |
| 卡诺图合并项数 | 2 的整数次幂 |
| 卡诺圈能否重叠 | **可以** |
| 无关项符号 | `×`（可作 0 或 1） |
| 对偶式变换 | `· ↔ +`，`0 ↔ 1`，变量不变 |
| 反演变换 | 对偶 + 变量取反 |

后续章节相关特性方程（此章简答3、4 涉及）：
- `D 触发器：Q* = D`
- `T 触发器：Q* = T ⊕ Q = TQ̄ + T̄Q`
- `JK 触发器：Q* = JQ̄ + K̄Q`
- `SR 触发器：Q* = S + R̄Q`，约束 `SR = 0`

## 易错点提醒
- **判断11陷阱**：最小项总数是 `2ⁿ`，不是 `n`（典型混淆）
- **判断12陷阱**："所有逻辑函数都能用卡诺图化简"过于绝对，但题库语境下卡诺图适用于变量数较少（≤4）的函数化简，考试按一般命题判断
- **判断13陷阱**：卡诺图方格**可被多个圈包围**（圈可重叠是化简关键）
- 求对偶式**不要改变变量本身**，求反函数才把变量取反；两种变换极易混淆
- 含无关项的化简题（简答5–7），必须把 `∑d(...)` 在卡诺图中标 `×`，并在能扩大圈时纳入为 1，不能扩大时丢弃为 0
