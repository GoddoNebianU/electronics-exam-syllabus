# 第1章 数字逻辑概论

> **题型分布**：判断 1–8、填空 1–7
> **重要度**：★★☆（基础概念题，题量适中，几乎全为送分题，务必拿满）

## 核心知识点

### 1. 数字信号与模拟信号
- 数字信号在**时间上和数值上都是离散的**（判断3 ✓）
- 二进制数只有 **0 和 1** 两个数码（判断1 ✓）
- 数字信号在传输处理时，对器件参数精度要求**比模拟信号低**（判断4 ✗，反向命题）
- 数字电路抗干扰能力强、精度要求低、便于存储与处理

### 2. 数制转换（必考填空）
- **十进制 → 二进制**：整数部分"除2取余，逆序排列"；小数部分"乘2取整，顺序排列"
- 典型结论（填空原题）：
  - `56 = 111000B`，`17 = 10001B`，`42 = 101010B`，`57 = 111001B`，`127 = 1111111B`，`43 = 101011B`
  - 小数：`0.625 = 0.101B`
- **十六进制 → 十进制**：按权展开。如 `3A.8H = 3×16 + 10 + 8/16 = 58.5`

### 3. 二—十进制编码（BCD 码）
- **编码**：在选定的一系列二值代码中，赋予每个代码一固定含义（判断2 ✓）
- **8421 BCD 码**：有权码，权值依次 8、4、2、1；每4位表示一位十进制数
  - 示例：`1001 → 9`，`0010 0101 → 25`，`0101 0110 → 56`，`1001 0011 → 93`
  - `127 → 0001 0010 0111`，`43 → 0100 0011`
- **余 3 码**：**无权码**，等于对应 8421 BCD 码加 `0011`，二者规则**相关**（判断6 ✗）
- **格雷码（Gray）**：**无权码**，相邻两个代码只有 1 位不同，用于减少转换错误

## 重点结论 / 公式速记

| 代码类型 | 是否有权 | 权值 / 特征 |
|----------|----------|-------------|
| 8421 BCD | 有权 | 8、4、2、1 |
| 余 3 码 | **无权** | = 8421 BCD + 0011 |
| 格雷码 | **无权** | 相邻码仅 1 位不同 |

- 十→二整数：除2取余逆序；十→二小数：乘2取整顺序
- BCD 码与纯二进制**不同**：BCD 每 4 位一组对应 1 位十进制，`93 ≠ 1001 0011B` 的数值，而是 BCD 形式表示的 93

## 易错点提醒
- **判断4陷阱**：把"精度要求低"颠倒为"高"，数字信号优点恰是降低精度要求
- **判断5陷阱**：把"8421有权 + 格雷码无权"混为一谈；格雷码必为无权码
- **判断6陷阱**：余3码虽无权，但其编码规则由 8421 BCD 加 3 得到，**并非无关**
- **判断7、8陷阱**：只要出现"余3码/格雷码 + 有权码"组合即为错
- BCD 码 `0110 0011` 表示十进制 63，而**不是**二进制数 0110 0011 的数值 99，二者不要混淆
