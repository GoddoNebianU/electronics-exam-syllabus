# 第5章 锁存器与触发器

> **题型分布**：判断 29–33、填空 27–29、简答 3–4、10
> **重要度**：★★★（特性方程必考简答，是时序章节的基础）

## 核心知识点

### 1. 锁存器 vs 触发器（填空33）
- **锁存器**：**电平敏感**（电平触发），使能信号有效期间输出随输入变化
- **触发器**：**边沿敏感**（边沿触发），仅在时钟有效沿瞬间状态变化
- **判断31陷阱**：触发器是**边沿**触发电路，原题说"电平触发"是错的
- **透明态与保持态**（填空34）：使能有效时输出跟随输入=**透明态**；使能无效时输出保持=**保持态**

### 2. 基本 RS 触发器（约束条件判断32、33）
- **由或非门构成**：约束条件是 `R、S` 不能**同时为 1**
- **由与非门构成**：约束条件是 `R̄、S̄`（低有效）不能**同时为 0**，即输入不能同时为 0
  - **判断32陷阱**：与非门构成时约束是"不能同时为 **0**"（原题说"不能同时为1"是错的，那是或非门）
  - **判断33陷阱**：原题"输入不能同时为0"对**与非门构成**的 RS 触发器是**正确**的；但若未说明结构，则需结合上拉/下拉约定判断
- 约束统一表达：`SR = 0`（不允许 S、R 同时有效）

### 3. 四种功能触发器特性方程（简答3、4 必考）
- **D 触发器**（数据/延迟）：
  ```
  Q* = D
  ```
- **T 触发器**（翻转）：填空27
  ```
  Q* = T ⊕ Q = TQ̄ + T̄Q
  ```
  - `T = 0`：状态**保持**不变
  - `T = 1`：每来一个时钟有效沿，输出**翻转**一次（判断29 ✓）
- **JK 触发器**（通用）：填空28
  ```
  Q* = JQ̄ + K̄Q
  ```
  - `J = K = 0`：保持
  - `J = K = 1`：每来一个时钟有效沿，输出**翻转**（计数状态）
  - `J = 1, K = 0`：置 1
  - `J = 0, K = 1`：置 0
  - JK 与 D 相比**优势**：克服了 RS 约束，功能更全，`J=K=1` 可合法翻转
- **SR 触发器**：
  ```
  Q* = S + R̄Q ，  约束 SR = 0
  ```

### 4. JK 触发器的派生应用（简答4）
- 两输入**相反**（`J = K̄`，即一个为 1 另一个为 0）→ 构成 **D 触发器**
- 两输入**相同**（`J = K`）→ 构成 **T 触发器**

### 5. 主从 JK 触发器一次翻转现象（填空29）
- **一次翻转现象**：发生在**时钟 CP = 1（高电平）期间**，JK 在此期间若发生变化，触发器只在 CP 下降沿前翻转一次，可能导致状态错误
- **克服方法**：改用**边沿 JK 触发器**（仅在边沿采样）

### 6. 异步置位/复位端（判断30）
- 异步置位端（`S̄D` / `PRE`）和异步复位端（`R̄D` / `CLR`）**不受时钟脉冲边沿触发控制**（判断30 ✓）
- 优先级**高于**同步输入（J/K/D/T），用于上电初始化或强制清零
- 低电平有效居多

## 重点结论 / 公式速记

| 触发器 | 特性方程 | 约束 |
|--------|----------|------|
| RS（基本） | `Q* = S + R̄Q` | `SR = 0` |
| D | `Q* = D` | 无 |
| T | `Q* = T ⊕ Q` | 无 |
| JK | `Q* = JQ̄ + K̄Q` | 无 |

- 锁存器 = 电平敏感；触发器 = 边沿敏感
- 与非门 RS：约束输入不能同时为 **0**
- 或非门 RS：约束输入不能同时为 **1**
- 异步置位/复位优先级最高，不受 CP 控制

## 易错点提醒
- **判断29**：T=1 时 T 触发器每来一个 CP 翻转一次 ✓（送分题）
- **判断31**：触发器是**边沿**触发，不是电平触发
- **判断32**：与非门 RS 触发器约束是"不能同时为 **0**"（注意是 0 不是 1）
- **判断33**：注意与 32 对比，二者结构不同约束不同，别答混
- 特性方程默写：D→最简，JK→`JQ̄ + K̄Q`，T→`T⊕Q`，是简答3的必背内容
- 填空28 `J = K = 1` 时翻转（不是保持）
- 主从 JK 一次翻转发生在 **CP=1 期间**（高电平期间），克服办法是改用**边沿**触发器
