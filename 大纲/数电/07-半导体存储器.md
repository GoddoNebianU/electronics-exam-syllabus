# 第7章 半导体存储器

> **题型分布**：判断 45–50、填空 36–40
> **重要度**：★★☆（纯概念+简单计算题，无大题，但容量扩展计算是填空重点）

## 核心知识点

### 1. 存储器分类（判断45、46、49、50）
- 按存取方式分两大类（判断46 ✓）：
  - **RAM（随机存取存储器）**：可读可写
  - **ROM（只读存储器）**：只能读出（或特定条件下写入）
- 按掉电是否保留数据分：
  - **易失性**：RAM（含 SRAM、DRAM）掉电后**数据丢失**（判断50 ✓）
  - **判断45陷阱**：SRAM 掉电后数据**会丢失**（原题说"不会丢失"是错的）
  - **非易失性**：ROM 掉电后数据**不丢失**（判断49 ✗，原题说"会丢失"是错的）

### 2. SRAM 与 DRAM（判断47，填空37）
- **SRAM（静态 RAM）**：用**触发器**存储数据，速度快、不需刷新、集成度低、功耗较大
- **DRAM（动态 RAM）**：利用**栅极电容**存储信息（填空37），因电容漏电需**定期刷新**（判断47 ✓）
  - 优点：集成度高、功耗低、成本低
  - 缺点：需刷新电路、速度较慢
- 主存（内存条）多用 DRAM，Cache 多用 SRAM

### 3. 存储器容量与地址/数据线（填空36、48）
- 容量表示 `M×N`：**M 个存储单元（字数）× N 位（每字位数）**
- **数据线条数 = N**（每字位数）
- **地址线条数 = log₂(M)**
- 典型换算（填空36）：
  - `16K×16`：16 根数据线，`log₂16K = 14` 根地址线
  - `512K×8`：8 根数据线，`log₂512K = 19` 根地址线
  - `128K×8`：8 位数据，`log₂128K = 17` 根地址线
- **判断48陷阱**：`256×4` 的 RAM 有 **4 条数据线**（填空48，原题说法正确 ✓）

### 4. 存储器扩展（填空38、39、40）
- **位扩展**：字数不变、增加数据位数
  - 用 `1024×1` 扩展成 `1024×8`：需 `8 / 1 =` **8 片**（地址线并联、数据线分别引出）（填空39）
- **字扩展**：位数不变、增加字数，需增加地址线做**片选译码**
  - 用 `256×4` 扩展成 `1024×8`：先位扩展 `4→8` 需 2 片，再字扩展 `256→1024` 需 4 倍 → 共 `2 × 4 =` **8 片**（填空38）
- **字位同时扩展**：综合上述
  - `1K×4` 组成 `4K×8`：位扩展 2 倍、字扩展 4 倍 → 共 `2 × 4 =` **8 片**，地址线中 `log₂4 =` **2 位**用作片选译码（填空40）

## 重点结论 / 公式速记

| 项 | 公式 / 结论 |
|----|-------------|
| 数据线数 | = 每字位数 N |
| 地址线数 | = `log₂M`（M 为字数） |
| 位扩展芯片数 | `目标位 / 芯片位` |
| 字扩展芯片数 | `目标字数 / 芯片字数` |
| 字位扩展芯片数 | 两者相乘 |
| 字扩展片选位 | `log₂(目标字数 / 芯片字数)` |

| 存储器类型 | 易失 | 刷新 | 用途 |
|-----------|------|------|------|
| SRAM | 是 | 否 | Cache |
| DRAM | 是 | **是** | 主存 |
| ROM | **否** | 否 | 固化程序 |

## 易错点提醒
- **判断45**：SRAM 掉电**会**丢失数据（易失）
- **判断49**：ROM 掉电**不**丢失数据（非易失）
- **判断50**：RAM 掉电**会**丢失数据
- **填空37**：DRAM 用**栅极电容**存储（不是触发器，那是 SRAM）
- **填空38**：`256×4 → 1024×8` 需 **8 片**（不是 4 片，位字都要扩展）
- **填空39**：`1024×1 → 1024×8` 需 **8 片**（纯位扩展）
- **填空40**：`1K×4 → 4K×8` 需 **8 片**，**2 位**地址做片选
- 容量计算注意 1K = 1024 = 2¹⁰，1M = 2²⁰，地址线位数用 log₂ 计算
- `256×4` 的 RAM 数据线数 = **4**（不是 256 或 log₂256）
