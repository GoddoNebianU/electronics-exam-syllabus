## 核心知识点

### 1. 存储器分类（判断 45–46、49–50）

#### 1.1 按存取方式分类（判断46 ✓）

- **RAM（随机存取存储器）**：可读可写，任意地址存取时间相同
- **ROM（只读存储器）**：数据固化，只能读出（或特定条件下写入）

#### 1.2 按掉电是否保留数据分类

| 类型 | 掉电后数据 | 包含 |
|------|-----------|------|
| **易失性** | **丢失** | RAM（SRAM、DRAM） |
| **非易失性** | **不丢失** | ROM |

- **判断45陷阱**：SRAM 是易失性存储器，掉电后数据**会丢失**。原题说"不会丢失"是**错的**
- **判断49陷阱**：ROM 是非易失性存储器，掉电后数据**不会丢失**。原题说"会丢失"是**错的**
- **判断50 ✓**：RAM 掉电后数据**会丢失**（无论 SRAM 还是 DRAM）

### 2. SRAM 与 DRAM 对比（判断 47，填空 37）

| 特性 | SRAM（静态 RAM） | DRAM（动态 RAM） |
|------|------------------|------------------|
| 存储元件 | **触发器** | **MOS 管栅极（寄生）电容** |
| 是否需刷新 | **否** | **是**（电容漏电，需定期刷新） |
| 速度 | 快 | 较慢 |
| 集成度 | 低 | 高 |
| 功耗 | 较大 | 低 |
| 成本 | 高 | 低 |
| 典型用途 | Cache（高速缓存） | 主存（内存条） |

- **判断47 ✓**：DRAM 通过电容存储数据，电荷会随时间泄漏，必须**定期刷新**以维持数据
- **填空37**：DRAM 利用 **MOS 管栅极（寄生）电容** 存储信息，需要定期刷新

### 3. 存储器容量与地址/数据线（填空 36、判断 48）

#### 3.1 容量表示法

- 容量表示 `M×N`：**M 个存储单元（字数）× N 位（每字位数）**
- **数据线条数 = N**（每字位数）
- **地址线条数 = $\log_2 M$**（字数对应的二进制编码位数）

#### 3.2 典型换算（填空 36）

| 容量 | 数据线 | 地址线 | 计算过程 |
|------|--------|--------|----------|
| 16K×16 | **16** 根 | **14** 根 | 16K = $2^{14}$ |
| 512K×8 | **8** 根 | **19** 根 | 512K = $2^{19}$ |
| 128K×8 | **8** 位 | **17** 根 | 128K = $2^{17}$ |

#### 3.3 判断 48

- **判断48 ✓**：256×4 位 RAM 有 **4 条数据线**（256 个字、每字 4 位），地址线为 $\log_2 256 = 8$ 条

### 4. 存储器扩展（填空 38–40，必考计算）

#### 4.1 位扩展（增加数据位数）

- 字数不变，增加每字位数
- 芯片地址线并联，数据线分别引出
- **填空39**：用 1024×1 位扩展成 1024×8 位 → 需 $8/1 =$ **8 片**

#### 4.2 字扩展（增加字数）

- 位数不变，增加字数，需增加地址线做**片选译码**

#### 4.3 字位同时扩展（填空 38、40）

- 总芯片数 = 字扩展倍数 × 位扩展倍数

| 扩展任务 | 字扩展 | 位扩展 | 总片数 | 片选位数 |
|----------|--------|--------|--------|----------|
| 256×4 → 1024×8（填空38） | 1024/256=4 倍 | 8/4=2 倍 | $4×2=$ **8 片** | $\log_2 4 = 2$ 位 |
| 1K×4 → 4K×8（填空40） | 4K/1K=4 倍 | 8/4=2 倍 | $4×2=$ **8 片** | $\log_2 4 =$ **2 位** |

- **填空40**：1K×4 位组成 4K×8 位需 **8 片**芯片，地址线中 **2 位**用作片选译码
  - 4K = $2^{12}$ 需 12 位地址，1K = $2^{10}$ 占用 10 位，剩 $12-10 = 2$ 位作片选

## 重点结论 / 公式速记

| 项 | 公式/结论 |
|----|-----------|
| 数据线数 | = 每字位数 N |
| 地址线数 | = $\log_2 M$（M 为字数） |
| 位扩展芯片数 | 目标位 / 芯片位 |
| 字扩展芯片数 | 目标字数 / 芯片字数 |
| 字位扩展总片数 | 字扩展倍数 × 位扩展倍数 |
| 字扩展片选位数 | $\log_2$(目标字数 / 芯片字数) |

| 存储器类型 | 易失 | 刷新 | 用途 |
|-----------|------|------|------|
| SRAM | 是 | 否 | Cache |
| DRAM | 是 | **是** | 主存 |
| ROM | **否** | 否 | 固化程序 |

**容量单位换算**：1K = $2^{10}$ = 1024，1M = $2^{20}$，1G = $2^{30}$
