## 核心知识点

### 1. DAC 与 ADC 功能辨析（判断 55–60 重点）

| 转换器 | 全称 | 功能（转换方向） |
|--------|------|------------------|
| **DAC** | 数模转换器 | **数字量 → 模拟量** |
| **ADC** | 模数转换器 | **模拟量 → 数字量** |

- **判断55 ✓**：ADC 能将模拟量转换为数字量
- **判断56陷阱**：DAC 是**数字量→模拟量**，原题说"模拟→数字"是**错的**
- **判断57 ✓**：A/D 取样将**时间上连续**的模拟量转换为**时间上断续**的模拟量（幅值仍连续）
- **判断58陷阱**：D/A 转换器将**数字→模拟**，原题说"模拟→数字"是**错的**
- **判断59陷阱**：DAC 功能是**数字→模拟**，原题说"模拟→数字"是**错的**
- **判断60 ✓**：ADC 功能是将模拟量转换为数字量

> ⚠️ **高频陷阱**：判断56、58、59 三题都是把 DAC 方向说反（模拟→数字），必须牢记 **DAC = 数→模**。

### 2. DAC 分辨率（填空 44–46）

- **分辨率定义**：最小输出电压（对应输入数字量仅 LSB=1）与最大输出电压之比
- 公式：分辨率 $= \frac{1}{2^n - 1}$，n 为位数

| 位数 | 分辨率（比值） | 满量程分数 |
|------|---------------|-----------|
| 8 位 | $1/(2^8-1) =$ **1/255** | $1/2^8 =$ **1/256**（填空45） |
| 10 位 | $1/(2^{10}-1) =$ **1/1023** | $1/2^{10} =$ **1/1024**（填空46） |

- **填空44**：8 位 DAC 分辨率 = **1/255**
- **填空45**：8 位 DAC 用最小量化单位表示满量程的 **1/256**
- **填空46**：10 位 DAC 用最小量化单位表示满量程的 **1/1024**

> 注意：$1/(2^n-1)$ 和 $1/2^n$ 是两种不同的表示法，前者是严格分辨率定义，后者是工程近似。填空45、46 要求是"满量程的分数"，答案为 $1/2^n$。

### 3. DAC 电路结构（填空 47）

#### 3.1 权电阻网络 DAC

- **缺点**：各位电阻阻值种类多、相差悬殊（如 $R, 2R, 4R, \ldots, 2^{n-1}R$），**难以集成制造**且精度难保证

#### 3.2 倒 T 型电阻网络 DAC（R-2R 梯形）

- **优点**：网络中只有 **R 和 2R 两种电阻**，**易于集成**；各支路电流恒定、**转换速度快**、精度高
- 是目前集成 DAC 中应用最广的电路形式

### 4. A/D 转换四步（填空 44）

A/D 转换过程分四步（顺序不可颠倒）：

$$\text{取样} \to \text{保持} \to \text{量化} \to \text{编码}$$

- **取样**：时间上连续→断续（需满足取样定理 $f_s \geq 2f_{max}$），幅值仍连续
- **保持**：保持取样值一段时间，供后续量化使用
- **量化**：将取样电压归类到有限个离散电平
- **编码**：将量化结果用二进制代码表示

### 5. DAC 功能与 A/D 四步（填空 44 综合）

- DAC 功能：将输入的**数字量**转化为与之成比例的**模拟量**输出
- A/D 四步：**取样**、保持、量化、**编码**

### 6. ADC 类型对比（填空 48–50）

| 类型 | 转换速度 | 抗干扰能力 | 特点 | 教材章节 |
|------|----------|-----------|------|----------|
| **并联比较型**（Flash） | **最快** | 一般 | n 位需 $2^n-1$ 个比较器，电路庞大 | §10.2 |
| **逐次逼近型**（SAR） | 中等（快） | 一般 | n 位需 **n 个时钟周期** | §10.2 |
| **双积分型** | **最慢** | **最强** | 对周期性干扰有极强抑制 | §10.2 |

- 速度排序：**并联比较型 > 逐次逼近型 > 双积分型**（填空48）
- **填空48**：
  - 逐次逼近型速度比双积分型**快**
  - 双积分型抗干扰能力比逐次逼近型**强**
  - 速度由快到慢：并联比较型、逐次逼近型、**双积分型**

### 7. 逐次逼近型 ADC 转换周期（填空 49）

- n 位逐次逼近型 ADC 完成一次转换至少需要 **n 个时钟周期**
- **填空49**：10 位逐次逼近型 ADC 需 **10 个时钟周期**

### 8. 双积分型 ADC 工作原理（填空 50）

- **第一阶段**：对**输入电压**进行**定时（正向）**积分（固定时间 $T_1$）
- **第二阶段**：对**基准电压**进行**反向（定值/定压）**积分（直至输出过零，时间 $T_2$ 正比于输入电压）

## 重点结论 / 公式速记

| 项 | 结论/公式 |
|----|-----------|
| DAC 功能 | **数字量 → 模拟量** |
| ADC 功能 | **模拟量 → 数字量** |
| DAC 分辨率 | $1/(2^n - 1)$ |
| 8 位 DAC 满量程分数 | **1/256** |
| 10 位 DAC 满量程分数 | **1/1024** |
| A/D 四步 | 取样 → 保持 → 量化 → 编码 |
| 逐次逼近 n 位周期数 | **n** |
| ADC 速度排序 | 并联比较 > 逐次逼近 > 双积分 |
| 权电阻网络缺点 | 电阻种类多、难集成 |
| 倒 T 型优点 | 只用 R 和 2R、易集成、速度快 |
| 双积分第一阶段 | 对输入电压**定时**积分 |
| 双积分第二阶段 | 对基准电压**反向定值**积分 |
