## 核心知识点

### 1. 时序电路概述（判断 34–44）

- **时序电路 = 组合电路 + 触发器（存储元件）**
- 输出不仅取决于当前输入，还与**电路原状态（历史状态）**有关（填空30）

#### 1.1 同步 vs 异步

| 特性 | 同步时序电路 | 异步时序电路 |
|------|-------------|-------------|
| 时钟 | 所有触发器**共用同一时钟 CP** | 无统一时钟 |
| 状态变化 | 同时发生 | 依赖输入信号和电路延迟 |
| 是否含触发器 | **是** | **也含触发器** |

- **判断35陷阱**：同步时序电路中各触发器时钟**相同**（不是"可以不同"），原题说"可以不同"是**错的**
- **判断38陷阱**：异步时序电路**也包含触发器**（区别在于无统一时钟，而非是否含触发器），原题说"不包含"是**错的**
- **判断40 ✓**：同步时序电路的时钟脉冲同时加到各触发器
- **判断42 ✓**：同步时序电路中所有触发器的状态变化受同一时钟信号控制
- **判断44 ✓**：异步时序电路无统一时钟，状态变化由输入信号和电路内部延迟共同决定

#### 1.2 竞争冒险对时序电路的影响

- **判断34陷阱**：竞争冒险**也会影响**时序电路（可能使触发器误翻转，破坏时序功能）。原题说"没有影响"是**错的**

#### 1.3 Mealy 型与 Moore 型（填空 32）

- **Mealy（米利）型**：输出取决于**当前输入**和**当前状态（现态）**（填空32，两个因素）
- **Moore（穆尔）型**：输出仅取决于当前状态

### 2. 三种等价描述（判断 39）

- **判断39 ✓**：**时序图、状态转换图、状态转换表**三者可描述同一时序电路，且**可相互转换**
- 分析题标准流程（简答 27–30）：驱动方程 → 状态方程 → 状态表 → 状态图 → 逻辑功能

### 3. 计数器（判断 36–37，填空 30–31）

- **模**：计数器一个计数循环中的状态数 = 能计数的最大脉冲个数（判断37 ✓）
- 由 n 个触发器构成的最大计数模值 = **$2^n$**（填空30）
- 4 位二进制加法器最多能计 **16** 个脉冲（填空30）

#### 3.1 模 60 计数器设计（填空 31）⭐ 修正

- 至少需要 **6 个触发器**（$2^5=32 < 60 \leq 64 = 2^6$）
- 若用 74160（十进制计数器）实现，需 **2 片**级联
  - ⚠️ **修正**：旧版大纲误写"6 片"，实际答案为 **2 片**（一片作个位模 10、一片作十位模 6）

#### 3.2 环形与扭环计数器（判断 36）

| 类型 | 构成 | 有效状态数 |
|------|------|-----------|
| 环形计数器 | n 位移位寄存器 | **n 个** |
| 扭环计数器（约翰逊） | n 位移位寄存器 | **2n 个** |

- **判断36陷阱**：环形计数器有效状态数是扭环的**一半**（n vs 2n），原题说"两倍"是**错的**

### 4. 集成计数器 74LVC161 / 74HC161 / 74160（简答 20–22）

#### 4.1 74LVC161 / 74HC161 引脚与功能

- 4 位二进制同步加法计数器（模 16）
- 引脚：`CP`（时钟）、$\overline{PE}$（同步并行置数）、$\overline{CR}$（异步清零，低有效）、`CET/CEP`（使能）、`TC`（进位输出）
- 功能优先级：$\overline{CR} > \overline{PE} > CET/CEP >$ 计数

#### 4.2 任意模计数器设计（简答 20–22）

- **反馈清零法**：计到目标值时触发 $\overline{CR}$（异步，可能有毛刺）
- **反馈置数法**（推荐，无毛刺）：计到目标值时触发 $\overline{PE}$，同步置入初值
- **级联法**：模 168 = 两片 161 级联（同步级联，高位片 CET/CEP 接低位片 TC）

#### 4.3 简答22：模 168 和模 11 计数器设计

**模 11（单片）**：状态序列 0000→…→1010，计到 1010 时 $\overline{PE} = \overline{Q_3 \cdot Q_1}$，置数 0000

**模 168（两片级联）**：
- 167 = 1010 0111₂（高位片 1010，低位片 0111）
- 同步级联：高位 CET/CEP 接低位 TC
- 反馈检测：$Y = Q_{13} \cdot Q_{11} \cdot Q_{02} \cdot Q_{01} \cdot Q_{00}$
- $\overline{PE} = \overline{Y}$，同时接两片，置数 0000

### 5. 寄存器与移位寄存器（判断 41、43，填空 35）

- **判断41 ✓**：寄存器主要功能是**暂时存储二进制数据**
- **填空35**：4 位数据寄存器在时钟有效沿同时存入 4 位并行数据：**并行输入、并行输出**
- **判断43 ✓**：移位寄存器既能存储数据，又能实现**串行↔并行转换**（串入并出、并入串出）

### 6. 综合题分析要点（综合题 1–2）

综合题由 555 定时器 + 74LVC161 + 74HC138/74HC153 组成：
1. **555 构成多谐振荡器**（无稳态，作 CP 源）
2. 计算 $v_{o1}$ 周期 $T \approx 0.7(R_A + 2R_B)C$
3. **161 计数器**：分析反馈逻辑确定模
4. **138 译码器 / 153 选择器**：根据 MN 不同输入，对应不同进制
5. 当 MN=00 时画状态转换图

## 重点公式 / 结论速记

| 项 | 结论 |
|----|------|
| 同步 vs 异步 | 同步共用 CP，异步无统一时钟 |
| n 触发器最大模 | $2^n$ |
| 4 位二进制计数范围 | 0–15（16 个脉冲） |
| 环形计数器有效状态 | $n$ |
| 扭环计数器有效状态 | $2n$ |
| Mealy 输出 | 依赖输入 + 现态 |
| Moore 输出 | 仅依赖现态 |
| 模 60 计数器触发器数 | **6 个** |
| 模 60 用 74160 实现 | **2 片**级联 |

- 74LVC161 功能优先级：$\overline{CR} > \overline{PE} > CET \cdot CEP$
- 任意模 M 设计：反馈清零（用 M 的二进制译码）或反馈置数（置 16−M）
