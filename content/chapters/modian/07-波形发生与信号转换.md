## 核心知识点

### 1. 正弦波振荡的平衡条件<sup>填空73–74</sup>

#### 1.1 平衡条件（维持等幅振荡）

| 条件 | 公式 |
|------|------|
| **幅度平衡条件** | $|AF| = 1$ |
| **相位平衡条件** | 环路总相移 $\Sigma\varphi = 2n\pi$（$2\pi$ 的整数倍） |

#### 1.2 起振条件（从小到大建立振荡）

| 条件 | 公式 |
|------|------|
| **起振幅度条件** | $|AF| > 1$（**严格大于**） |
| **起振相位条件** | $\Sigma\varphi = 2n\pi$ |

- **填空73**：幅度平衡 $|AF|=1$；相位平衡 $\Sigma\varphi = 2n\pi$
- **填空74**：起振幅度 $|AF| > 1$；相位 $2\pi$ 的整数倍
- 从起振到稳幅：随幅度增大，器件进入非线性区，$A$ 自动下降，直到 $|AF|=1$ 稳定

### 2. 正弦波振荡电路组成<sup>振荡题3</sup>

- **放大电路**：提供足够增益
- **选频网络**：决定振荡频率（单一频率）
- **正反馈网络**：满足相位条件
- **稳幅环节**：保证 $|AF|=1$（振幅稳定）

### 3. RC 正弦波振荡电路<sup>填空75、振荡题1–4</sup>

*图：RC 文氏电桥振荡器——RC 串并联选频网络接同相端（正反馈），$R_f/R_1$ 分压接反相端（稳幅负反馈）*

#### 3.1 基本特点

- 适用于产生**低频**信号（频率范围一般 1Hz ~ 1MHz）<sup>填空75</sup>
- 选频网络由 **R 和 C** 组成（文氏电桥/串并联 RC 网络）

#### 3.2 文氏电桥振荡频率

$$f_0 = \frac{1}{2\pi RC}$$

#### 3.3 文氏电桥关键参数

- 在 $f_0$ 处：反馈系数 $F = 1/3$，相移为 0（满足相位条件）
- 为满足起振条件 $|AF| > 1$：要求 $A > 3$
- 同相放大倍数 $A = 1 + R_f/R_1 > 3$ → **$R_f > 2R_1$**（起振电阻条件）

### 4. LC 正弦波振荡电路（对比了解）

- 选频网络由 L 和 C 组成，适用于**高频**信号（> 1MHz）
- 常见形式：变压器反馈式、电感三点式（哈特莱）、电容三点式（考毕兹）
- 题库以 RC 文氏电桥为主

### 5. 方波发生器<sup>填空76、78</sup>

#### 5.1 组成

- 由 **滞回比较器（电压比较器）** + **RC 积分（反馈）网络** 组成<sup>填空76</sup>

#### 5.2 工作原理

- 输出经 RC 反馈到反相端，电容电压在两个门限之间反复充放电，输出方波

#### 5.3 参数决定因素<sup>填空78</sup>

- 输出电压幅度取决于**稳压管限幅值**（或电源电压）
- 振荡频率取决于**电阻 R 和电容 C**的参数（以及比较器阈值）

### 6. 三角波发生器<sup>填空77</sup>

- 由 **方波发生器** + **积分器** 两部分级联构成<sup>填空77</sup>
- 方波经积分得到三角波
- 若要锯齿波，让积分器充放电时间常数不等

### 7. 文氏电桥题常见问题<sup>振荡题1–4</sup>

| 题号 | 核心问题 | 关键答案 |
|------|----------|----------|
| 题1 | 热敏电阻 $R_t$ 温度系数 + 求 $I_t$/$R_t$/$V_O$ | **负**温度系数（电流大→升温→阻值降→负反馈增强→降增益稳幅） |
| 题2 | 标运放同相/反相端 + $R_p+R_2$ 起振下限 | RC 网络接**同相端**；分压电阻接**反相端**；$R_p+R_2 > 2R_1$ |
| 题3 | 选频/负反馈网络组成 + $R_t$ 与 $R_1$ 关系 + 温度系数 | 选频=RC 串并联；负反馈=$R_t$、$R_1$；稳定时 $R_t = 2R_1$；**负**温度系数 |
| 题4 | 文氏桥不振荡找错并改正 | 常见错误：反馈极性接反或 $A \leq 3$ |

| 项目 | 公式/结论 |
|------|----------|
| 振荡幅度平衡 | $|AF| = 1$ |
| 振荡相位平衡 | $\Sigma\varphi = 2n\pi$ |

## 易错点提醒（对应题号）

| 题号 | 陷阱描述 | 正确结论 |
|------|----------|----------|
| **填空73–74** | 平衡条件 vs 起振条件 | 平衡：$|AF|=1$；起振：$|AF|>1$（**严格大于**） |
| **填空75** | RC 振荡适用频率 + 选频网络 | **低频**；选频网络由 **R 和 C** 组成 |
| **填空76** | 方波发生器组成 | **滞回比较器** + **RC 积分网络** |
| **填空77** | 三角波发生器组成 | **方波发生器** + **积分器** |
| **填空78** | 方波幅度/频率取决于什么 | 幅度=**稳压管**；频率=**R 和 C** |
| **振荡题1** | 热敏电阻温度系数 | **负**温度系数（自动稳幅） |
| **振荡题2** | 文氏桥运放接线 | RC 选频网络接**同相端**（正反馈）；分压电阻接**反相端**（负反馈稳幅） |
| **振荡题3** | 稳定时 $R_t$ 与 $R_1$ 关系 | $R_t = 2R_1$（此时 $A = 3$，$|AF| = 1$） |
